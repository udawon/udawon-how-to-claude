# 변경 이력: Claude Code 버전별 주요 업데이트

Claude Code는 스마트폰 앱처럼 지속적으로 업데이트됩니다. 스마트폰에서 "앱 업데이트" 알림이 오면 새 기능이 추가되고 버그가 고쳐지듯이, Claude Code도 정기적으로 새로운 기능이 추가되고 문제가 수정됩니다. 이 문서는 각 버전에서 무엇이 바뀌었는지를 비개발자도 이해할 수 있게 정리합니다.

---

## 최신 버전: 2.1.72

### 새로운 기능

| 기능 | 설명 |
|---|---|
| `/copy` 파일 저장 | 복사 메뉴에서 `w` 키로 선택한 내용을 파일에 직접 저장 (SSH 환경에서 유용) |
| `/plan` 설명 추가 | `/plan 인증 버그 수정` 처럼 계획에 설명을 붙일 수 있음 |
| `ExitWorktree` 도구 | 워크트리 세션을 종료하는 도구 추가 |
| 자동 허용 명령 확장 | `lsof`, `pgrep`, `tput`, `ss`, `fd`, `fdfind` 명령이 자동 승인 목록에 추가됨 |

### 성능 개선

- 모델 사용 시 노력 수준을 낮음/보통/높음(○ ◐ ●)으로 단순화
- 번들 크기 약 510KB 감소
- Bash 명령 파싱 속도 향상 및 메모리 누수 수정

### 설정

- `CLAUDE_CODE_DISABLE_CRON` 환경변수로 세션 중 예약 작업 중단 가능
- `/config` 명령 개선: Escape로 취소, Enter로 저장, Space로 토글
- `/reload-plugins` 명령 추가: 재시작 없이 플러그인 변경 적용

### 주요 버그 수정

- 음성 입력 지연, "말이 감지되지 않음" 오류, 오래된 전사 텍스트 문제 수정
- 샌드박스 파일 쓰기 및 출력 리다이렉션 권한 문제 수정
- 팀 에이전트가 리더의 모델을 상속하지 않던 문제 수정
- "항상 허용" 저장이 이후에 매칭되지 않던 문제 수정
- 세션 충돌(U+2028/U+2029 문자) 수정

## 버전 2.1.71

### 새로운 기능

| 기능 | 설명 |
|---|---|
| `/loop` 명령 | 반복 실행 설정 (예: `/loop 5m 배포 상태 확인`으로 5분마다 자동 실행) |
| 음성 Push-to-Talk | 스페이스바로 음성 활성화 (키 바인딩 변경 가능) |
| 크론 예약 도구 | 세션 내에서 반복 예약 프롬프트 설정 |

### 성능 및 안정성

- 장시간 세션에서 입력 멈춤 현상 수정
- macOS에서 CoreAudio 초기화로 인한 5~8초 지연 수정
- 플러그인 설치가 여러 Claude Code 인스턴스 사용 시 사라지던 문제 수정

---

## 버전 2.1.70

### 버그 수정

- 서드파티 게이트웨이(`ANTHROPIC_BASE_URL`) 사용 시 400 오류 수정
- 커스텀 Bedrock 프로필에서 "effort 파라미터 미지원" 오류 수정
- Windows/WSL에서 클립보드가 한국어·이모지 등 비ASCII 문자를 깨뜨리던 문제 수정
- Windows에서 VS Code 창이 여러 개 열리던 문제 수정

## 버전 2.1.69

### 새로운 기능

| 기능 | 설명 |
|---|---|
| `/claude-api` 스킬 | Claude API 애플리케이션 구축 도우미 |
| 음성 STT 언어 확장 | 10개 언어 추가로 총 20개 언어 지원 |
| 에이전트 이름 표시 | 터미널 제목에 현재 에이전트 이름 표시 |
| `/reload-plugins` | 재시작 없이 플러그인 재로드 |
| Bash 빈 줄에서 Ctrl+U | Bash 모드 종료 |

### 고급 설정

- `includeGitInstructions` 설정 및 `CLAUDE_CODE_DISABLE_GIT_INSTRUCTIONS` 환경변수 추가
- `InstructionsLoaded` 훅 이벤트 추가
- `${CLAUDE_SKILL_DIR}` 변수 지원
- macOS에서 Go 프로그램(`gh`, `gcloud`, `terraform`)을 위한 약한 네트워크 격리 옵션 추가

### 보안 수정

- 중첩 스킬이 gitignore된 디렉토리에서 로드되던 문제 수정
- MCP 신뢰 대화상자가 모든 서버를 자동 활성화하던 문제 수정
- macOS 키체인 손상 문제 수정

## 버전별 개선 주제 요약

Claude Code는 출시 이후 지속적으로 다음 영역을 개선해 왔습니다.

| 주제 | 내용 |
|---|---|
| 메모리·성능 | 장시간 세션에서의 메모리 누수 지속 수정, 시작 속도 개선 |
| 보안 | 권한 처리 및 프롬프트 주입 방어 강화 |
| 사용자 경험 | UI 개선, 새 명령어 추가, 키보드 단축키 확장 |
| 크로스 플랫폼 | Windows, macOS, Linux, SSH 환경별 버그 수정 |
| 플러그인 시스템 | 마켓플레이스, 설치, 훅 기능 지속 개선 |
| 음성 기능 | 지원 언어 확장, 음성 입력 품질 개선 |
| 훅 시스템 | 이벤트 유형 및 기능 확장 |

## 예시 케이스

### 상황 1: 현재 버전 확인하기

```bash
claude --version
```

최신 기능을 사용하려면 `claude update`로 업데이트하세요.

### 상황 2: 업데이트 후 동작이 달라졌을 때

변경 이력을 확인해 어떤 기능이 바뀌었는지 확인하세요. 특히 비용 보고 방식이나 모델 기본값이 바뀔 수 있습니다.

### 상황 3: 특정 버전의 기능이 언제 추가됐는지 알고 싶을 때

이 문서에서 기능명으로 검색하거나 [공식 changelog](https://code.claude.com/docs/en/changelog)에서 전체 이력을 확인하세요.
