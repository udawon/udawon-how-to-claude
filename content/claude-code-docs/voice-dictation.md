## 이게 뭔가요?

음성 받아쓰기(Voice Dictation)는 **키보드로 타이핑하는 대신 말로 Claude Code에게 지시하는 기능**입니다.

카카오톡 음성메시지를 생각해보세요. 채팅창에서 마이크 버튼을 누르고 말하면 상대방에게 음성이 전달되죠? Claude Code의 음성 받아쓰기도 비슷합니다 — **Space 키를 꾹 누르고 말하면, 말한 내용이 텍스트로 변환되어 Claude에게 전달**됩니다. 차이점은 음성 파일이 아니라 **텍스트로 바뀐다**는 것입니다.

핵심 포인트:
- Push-to-talk(누르고 말하기) 방식 — Space 키를 누르고 있는 동안만 녹음
- 한국어, 일본어, 영어 등 **20개 언어** 지원
- 코딩 용어(함수명, 변수명 등)에 최적화된 음성 인식

## 왜 알아야 하나요?

**알면 좋은 점:**
- 긴 설명이나 복잡한 요청을 **타이핑 없이 빠르게** 전달 가능
- 손이 피곤할 때, 생각을 정리하면서 말로 지시할 수 있음
- 타이핑과 음성을 **같은 메시지 안에서 섞어서** 사용 가능
- 코딩 관련 단어(React, useState, API 등)를 정확하게 인식

**모르면 불편한 점:**
- 복잡한 요청을 길게 타이핑해야 하는 번거로움
- 손목이 아플 때도 키보드에 의존해야 함
- 빠르게 아이디어를 전달하고 싶을 때 타이핑 속도에 제한됨

## 어떻게 하나요?

### 방법 1: /voice 명령어로 활성화

**1단계: 음성 받아쓰기 켜기**

Claude Code 터미널(키보드로 명령어를 입력하는 화면)에서 입력:
```
/voice
```

처음 사용할 때 마이크 접근 권한을 요청할 수 있습니다. "허용"을 눌러주세요.

**2단계: Space 키 꾹 누르고 말하기**

- Mac: `Space` 키를 꾹 누른 상태에서 말하기
- Windows: `Space` 키를 꾹 누른 상태에서 말하기

손을 떼면 녹음이 끝나고, 말한 내용이 텍스트로 변환되어 입력창에 나타납니다.

**3단계: 확인 후 전송**

변환된 텍스트를 확인하고, 필요하면 수정한 뒤 Enter를 눌러 전송합니다.

<div class="example-case">
<strong>예시: 복잡한 요청을 음성으로 전달하기</strong>

개발자 김서연씨는 긴 리팩토링 요청을 해야 합니다. 타이핑하면 2분 걸릴 내용을 `/voice`를 켜고 Space 키를 누른 채 말합니다:

"지금 UserService 클래스에서 데이터베이스 직접 호출하는 부분을 전부 Repository 패턴으로 바꿔줘. 그리고 에러 처리도 try-catch로 감싸고, 각 메서드에 JSDoc 주석도 추가해줘."

20초 만에 상세한 요청이 텍스트로 변환되어 Claude에게 전달됩니다.

</div>

### 방법 2: Push-to-talk 키 변경

Space 키가 불편하다면 다른 키로 바꿀 수 있습니다.

```
/voice --key tab
```

위 명령은 Tab 키를 Push-to-talk 키로 변경합니다. 자주 Space를 누르는 작업(코드 편집 등)과 충돌할 때 유용합니다.

### 타이핑과 음성 섞어 쓰기

같은 메시지에서 타이핑과 음성을 번갈아 사용할 수 있습니다:

1. 먼저 키보드로 "src/components/Header.tsx 파일에서" 입력
2. Space 키를 누르고 "네비게이션 메뉴를 반응형으로 바꿔줘, 모바일에서는 햄버거 메뉴로 나오게" 라고 말하기
3. 다시 키보드로 추가 내용 수정
4. Enter로 전송

<div class="example-case">
<strong>예시: 코드 파일명은 타이핑, 설명은 음성으로</strong>

백엔드 개발자 이준호씨는 파일 경로처럼 정확해야 하는 부분은 키보드로 타이핑하고, 요청 사항 설명은 말로 합니다. "api/routes/auth.ts 파일을" 타이핑한 뒤 Space를 누르고 "로그인 실패 시 에러 메시지를 더 구체적으로 바꿔줘, 비밀번호 틀렸는지 아이디가 없는지 구분해서 알려주게"라고 말합니다.

</div>

<div class="example-case">
<strong>실전 케이스: 회의 직후 아이디어를 바로 구현 지시</strong>

"방금 회의에서 결정된 내용인데, 결제 페이지에서 쿠폰 입력란을 추가하기로 했어. 쿠폰 코드를 입력하면 실시간으로 할인 금액이 표시되게 해줘. 그리고 유효하지 않은 쿠폰이면 빨간색으로 에러 메시지 보여주고."

타이핑했으면 3분 걸렸을 요청을 30초 만에 전달합니다.

## 주의할 점

- **claude.ai 계정 필요** — 음성 받아쓰기는 claude.ai 계정으로 로그인한 상태에서만 사용 가능합니다. API 키 인증으로는 사용할 수 없습니다.
- **마이크 권한 허용** — 처음 사용 시 운영체제(Mac/Windows)에서 마이크 접근 권한을 요청합니다. 허용하지 않으면 동작하지 않습니다.
  - Mac: 시스템 설정 > 개인정보 보호 및 보안 > 마이크
  - Windows: 설정 > 개인 정보 > 마이크
- **인터넷 연결 필요** — 음성 인식은 서버에서 처리되므로 인터넷이 끊기면 작동하지 않습니다.
- **주변 소음 주의** — 조용한 환경에서 사용해야 인식률이 높습니다. 카페처럼 시끄러운 곳에서는 오인식이 발생할 수 있습니다.
- **한국어 지원** — 한국어, 영어, 일본어, 중국어, 프랑스어, 독일어, 스페인어 등 20개 언어를 지원합니다. 별도 언어 설정 없이 말하는 언어를 자동 인식합니다.
- **코딩 용어 인식** — "React", "useState", "npm install" 같은 코딩 용어에 최적화되어 있어 일반 음성 인식보다 정확합니다.

## 정리

- 음성 받아쓰기는 **Space 키를 누르고 말하면 텍스트로 변환**되어 Claude에게 전달하는 기능입니다.
- `/voice`로 활성화하며, 타이핑과 음성을 같은 메시지에서 섞어 쓸 수 있습니다.
- 한국어 포함 20개 언어를 지원하고, 코딩 용어 인식에 최적화되어 있습니다.
