# 제로 데이터 보존(ZDR): 대화 내용을 서버에 남기지 않는 방법

ZDR(Zero Data Retention)은 마치 **기억을 지우는 회의실**과 같습니다. 회의가 끝나면 모든 메모가 즉시 파기되어 서버 어디에도 남지 않는 것처럼, 여러분의 코드와 대화가 Anthropic 서버에 보존되지 않습니다.

---

## ZDR이란?

ZDR이 활성화되면, Claude Code 세션에서 주고받은 내용(프롬프트와 AI 응답)이 처리 직후 Anthropic 서버에서 삭제됩니다. 법률 준수나 악성 사용 방지를 위한 경우를 제외하고는 어떤 데이터도 남지 않습니다.

### ZDR 사용 가능 조건

- **Claude for Enterprise** 플랜 전용
- Anthropic 계정팀에 요청해야 활성화 가능
- **조직 단위로 설정** (조직마다 별도로 신청)

---

## ZDR 범위

### 보호되는 것

Claude Code에서 터미널을 통해 주고받는 **모든 AI 추론 호출**이 ZDR의 보호를 받습니다. 어떤 Claude 모델을 사용하든 동일하게 적용됩니다.

### 보호되지 않는 것

ZDR이 켜져 있어도 다음 항목은 일반 보존 정책이 적용됩니다.

| 항목 | 이유 |
|---|---|
| claude.ai 웹 채팅 | 웹 인터페이스 대화는 ZDR 대상 아님 |
| Cowork 세션 | 별도 제외 대상 |
| Claude Code Analytics | 프롬프트/응답은 저장 안 하지만, 이메일·사용 통계 등 메타데이터는 보존 |
| 사용자 및 좌석 관리 | 관리 데이터(이메일, 좌석 배정 등)는 일반 정책 적용 |
| 서드파티 연동 | MCP 서버 등 외부 도구의 데이터는 해당 서비스 정책 적용 |

---

## ZDR 활성화 시 사용 불가 기능

서버에 대화 내용을 저장해야만 작동하는 기능은 자동으로 비활성화됩니다.

| 비활성화 기능 | 이유 |
|---|---|
| Claude Code on the Web (웹 실행) | 서버에 대화 기록 보존 필요 |
| Desktop 앱의 원격 세션 | 프롬프트·응답 포함 지속적 세션 데이터 필요 |
| 피드백 제출 (`/feedback`) | 대화 데이터를 Anthropic으로 전송 |

이 기능들은 터미널에서 사용 시 오류 메시지로 차단됩니다.

---

## 정책 위반 시 데이터 보존

ZDR이 켜져 있더라도, 법률 요건 또는 사용 정책 위반이 감지된 경우 Anthropic은 관련 데이터를 **최대 2년**까지 보관할 수 있습니다.

---

## ZDR 신청 방법

1. Anthropic 계정팀에 연락
2. 계정팀이 내부 검토 요청 제출
3. Anthropic이 자격 확인 후 조직에 ZDR 활성화
4. 모든 활성화 작업은 감사 로그에 기록됨

> **중요**: 같은 계정 아래 새 조직을 만들어도 ZDR이 자동 적용되지 않습니다. 새 조직마다 별도로 신청해야 합니다.

---

## 기존 API ZDR에서 Enterprise로 전환

기존에 종량제 API 키로 ZDR을 사용하던 경우, Claude for Enterprise로 전환하면 다음 이점을 누릴 수 있습니다.

| 추가 기능 | 설명 |
|---|---|
| 사용자별 비용 제한 | 팀원 한 명이 비용을 과다 사용하지 못하게 제한 |
| Analytics 대시보드 | 팀 전체 사용 현황 시각화 |
| 서버 관리 설정 | 조직 정책을 일괄 적용 |
| 감사 로그 | 모든 사용 이력 추적 |

전환 시 ZDR은 그대로 유지됩니다.

## 예시 케이스

### 상황 1: 의료 소프트웨어 개발사
환자 데이터와 관련된 코드를 Claude Code로 개발하는 경우, ZDR을 활성화하면 코드와 대화가 Anthropic 서버에 남지 않아 HIPAA 등 의료 개인정보 규정 준수에 도움이 됩니다. BAA(Business Associate Agreement)도 ZDR과 함께 사용 가능합니다.

### 상황 2: 금융 기관
은행이나 핀테크 기업이 내부 금융 로직이 담긴 코드를 Claude Code로 개선할 때, ZDR을 통해 소스코드가 외부에 보존되지 않는다는 점을 규정 준수 팀에 증명할 수 있습니다.

### 상황 3: 정부 기관
기밀 소프트웨어를 개발하는 정부 조직에서 Claude Code를 도입할 때, ZDR은 "AI 도구 사용 시 코드가 서버에 저장되는가?"라는 감사 질문에 명확히 대답할 수 있게 해줍니다.
