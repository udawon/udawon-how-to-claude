## 이게 뭔가요?

**VidIQ**(비디큐)는 YouTube 크리에이터들이 가장 많이 쓰는 채널 분석 SaaS(Software as a Service, 클라우드 기반 소프트웨어)입니다. 영상 키워드 점수, 검색량, 경쟁 강도, 트렌드 모멘텀, 본인 채널 성장 추이를 한곳에서 봅니다.

이 영상은 VidIQ가 **MCP 서버**(MCP는 AI에게 외부 도구와 직접 대화할 통로를 열어주는 표준 규격)를 출시해 Claude에 직접 연결되는 신규 커넥터를 시연합니다. 이제 사용자는:

> Claude 채팅창을 떠나지 않고, Claude에게 한국어/영어로 "내 니치에서 이번 주에 뜬 키워드 알려줘"라고만 말하면 → Claude가 VidIQ를 호출해서 분석 결과를 직접 가져옴.

비유하자면 이렇습니다.

> **기존 방식** = 매주 VidIQ 사이트 들어가서 키워드 검색 → CSV 다운로드 → 엑셀로 정리 → 다시 ChatGPT나 Claude에 붙여넣기 → 분석 요청.
> **MCP 커넥터 방식** = 비서가 옆에 앉아서 "이번 주 트렌드 뽑아와봐" 한마디면 끝.

여기서 또 하나의 핵심 개념은 **Claude Project**(클로드 프로젝트)입니다. Claude의 "작업 폴더"라고 보면 됩니다. 한 프로젝트 안에 지시문, 참고 파일, 가이드라인을 모아두면 그 프로젝트에서 새로 시작하는 모든 채팅이 그 정보를 자동으로 기억합니다. 사람으로 치면 "내 책상 위에 항상 두는 메모 묶음"입니다.

## 왜 알아야 하나요?

### 1인 크리에이터·1인 유튜버에게

매주 콘텐츠를 만드는 사람의 가장 큰 고민은 "다음 영상 뭐 만들지?"입니다. VidIQ는 답을 가지고 있지만, 분석 결과를 본인 콘텐츠 전략에 적용하려면 또 한 번 머리를 써야 합니다.

이 워크플로우를 알면:

- "이번 주 내 니치에서 뜨는 키워드 + 그중 내 채널 평균 대비 잘 나올 가능성이 높은 주제" 같은 복합 질문을 한 번에 처리
- VidIQ → 엑셀 → ChatGPT → 다시 정리하는 4단계 워크플로우가 1단계로 압축
- 매주 export한 본인 채널 리포트를 Claude Project에 누적하면 **본인 채널 평균 대비 outlier(이상치, 평균보다 비정상적으로 잘 나오는 영상) 판정 정확도가 점점 올라감**

### 영상 화자(Arman)의 검증

영상 화자는 "AI 도구로 1인 1천만 달러 회사 만들기"를 목표로 하는 컨설턴트입니다. 200개 회사 자문 경험을 바탕으로 이 워크플로우를 본인 채널에 적용 중. 영상 자체가 "방금 팝업 떠서 즉흥으로 찍은" 형식인데도 구체성이 매우 높습니다.

## 어떻게 하나요?

### 사전 준비

1. **VidIQ 계정**: 무료 또는 유료 (사용 가능한 기능은 플랜에 따라 다름)
2. **VidIQ 브라우저 확장 프로그램** 설치
3. **Claude 계정** (Pro 이상 권장 — Connectors 기능 사용)

### 방법 1: 자동 팝업으로 연결 (가장 쉬움)

VidIQ 계정 + 브라우저 확장이 모두 있으면, Claude를 열었을 때 **자동으로 "VidIQ 커넥터를 연결하시겠습니까?" 팝업이 뜸**. 영상 화자도 "다른 일 하려고 Claude 열었다가 우연히 팝업 보고 즉흥으로 영상 찍었다"고 합니다.

1. 팝업의 "Connect" 버튼 클릭
2. OAuth(권한 위임 표준) 팝업에서 본인 VidIQ 계정으로 로그인 한 번
3. 끝

### 방법 2: 수동으로 추가 (팝업 거절했거나 안 떴을 때)

1. Claude → **Connectors** 페이지 진입
2. **Browse** → 스크롤해서 **Custom**(커스텀) 카테고리에서 **VidIQ for Claude** 찾기
3. **Configure**(설정) 클릭
4. 원클릭 OAuth 인증

### 방법 3: 채팅창에서 즉시 호출

설정이 끝났으면 사용은 매우 간단:

1. Claude에서 **새 채팅** 시작
2. 채팅창 좌측 **+** 버튼 클릭
3. **Connectors** 선택
4. **Add from VidIQ for Claude** 클릭
5. 미리 정의된 워크플로우 중 선택 (예: **Niche Evaluation**, **Content Gap Analysis**)
6. Claude가 미리 만들어둔 프롬프트 템플릿이 나옴 → 본인 키워드만 입력 → Enter

<strong>예시: "vibe coding for founders" 키워드 평가 결과</strong>

영상에서 화자가 실제로 던진 키워드와 결과:

- **검색량**: "vibe coding for founders" 자체는 검색량 거의 없음 → 너무 긴 롱테일
- **상위 키워드 "vibe coding"**: 월 100만 검색 이상, 트렌드 폭발 중
- **키워드 점수**: 91/100 (높음)
- **경쟁도**: 상위 키워드는 high, 롱테일 변형은 low
- **모멘텀**: 사람들이 많이 올리는 중이라 "declining" 신호 (포화 시작)
- **각도(Angle) 제안**: case study(사례 연구) 포지션 추천
- **Verdict**: "Good to go"

Claude가 단순 키워드 점수만 알려주는 게 아니라, **롱테일이 너무 길면 상위 키워드 추천**, 모멘텀까지 종합해 의사결정용 점수카드를 던져줍니다.

## Claude Project와 결합한 자동화 워크플로우

영상의 진짜 인사이트는 여기부터입니다. VidIQ 커넥터 단독으로 쓰는 게 아니라, **Claude Project 안에 누적된 본인 채널 데이터**와 결합하는 패턴.

### Project에 넣어둘 4종 자료

1. **Agent Instructions**(에이전트 지시문): "콘텐츠 전략 요청 시 반드시 VidIQ 커넥터로 키워드/콘텐츠 갭 조회 후 답변할 것"
2. **본인 니치 트랜스크립트 스크랩**: 같은 분야 인기 영상의 자막 모음 (Claude가 톤·구성 패턴 학습)
3. **매주 export한 자체 VidIQ 리포트**: 본인 채널의 주차별 성과 데이터
4. **가이드라인**: 본인의 콘텐츠 원칙(브랜드 톤, 다루지 않는 주제 등)

### 매주 반복 가능한 질문 패턴

```
이번 주 내 콘텐츠 아이디어 후보 [A, B, C, D, E]가 있어.
VidIQ 커넥터로 각 키워드의 모멘텀과 outlier 가능성을 조회해서,
이 중 어느 게 가장 트렌드와 맞고, 내 채널 평균 대비 outlier가 될 확률이 높은지
순위 매겨줘.
```

이 한 문장으로 매주의 콘텐츠 우선순위 결정이 자동화됩니다.

<div class="example-case">
<strong>실전 케이스: 한국어 1인 유튜버의 매주 콘텐츠 회의 자동화</strong>

상황: 약사 김씨가 약학·건강 관련 1인 채널을 운영. 매주 일요일 저녁 "다음 주 영상 뭐 찍지" 결정에 2시간 소요.

기존 방식:
- VidIQ에서 키워드 5개 검색 (30분)
- 본인 채널 분석 (20분)
- 경쟁 채널 영상 5개 시청 (40분)
- 메모 정리 + 결정 (30분)
- **총 2시간**

Claude Project + VidIQ 커넥터 방식:
- "complete-medications" 프로젝트에 매주 본인 채널 VidIQ 리포트 export 업로드
- 일요일 저녁 채팅: "이번 주 후보 5개: [영양제 비교, 두통약 차이, 어린이 해열제, 멜라토닌 부작용, 위장약 복용 시간]. 모멘텀과 outlier 가능성으로 순위 매겨줘"
- Claude가 VidIQ 자동 호출 → 5개 키워드 분석 + 본인 채널 데이터 비교 → 순위 + 근거 + 추천 각도까지 회신
- **총 10~15분**
</div>

<strong>실전 케이스: 콘텐츠 갭(Content Gap) 분석으로 작은 채널이 어떻게 뜨는지 학습</strong>

상황: "vibe coding basics for non-technical founders" 분야에 진입하려는 신규 채널.

Content Gap Analyzer 워크플로우 결과:
- 작은 채널(구독자 1만 이하) 중 breakout(폭발 성장) 영상 찾아냄

  - 썸네일 스타일
  - 영상 길이 분포
  - 후크(영상 첫 5초 패턴)
  - 제목 작성 공식
- "이 갭에 진입하면 X% 확률로 outlier 가능"이라는 점수 회신

작은 채널은 알고리즘이 적게 밀어줘서 "이미 잘 만들어진 영상도 묻히는" 위험이 큰데, 큰 채널이 안 다룬 갭을 정확히 짚어주는 게 핵심.

## 주의할 점

- **VidIQ 유료 플랜 기능 차이**: 일부 분석은 무료 플랜에서 제한될 수 있음. 영상에서 가격이나 플랜별 차이는 명시되지 않음 — 본인 계정 확인 필요.
- **자동 호출은 강제 명시 필요**: Project Instructions에 "콘텐츠 전략 답변 전 반드시 VidIQ 호출"을 넣지 않으면 Claude가 자체 추론으로만 답변할 수 있음.
- **outlier 판정의 정확도**: 본인 채널 데이터가 적으면 "평균 대비 outlier" 기준이 불안정. 영상 화자도 "매주 export 누적해서 정확도 올리는 중"이라고 함.
- **모든 데이터가 영어 기준**: VidIQ가 한국어 키워드도 분석하지만, 영어권 데이터가 압도적으로 많음. 한국어 채널은 결과를 보정해서 해석 필요.
- **본인 채널 데이터를 Project에 업로드할 때 개인정보 주의**: 시청자 인구통계, 수익 정보 등은 민감 데이터. 공유 가능한 수준만 업로드.

## 정리

- VidIQ MCP 커넥터로 **Claude 채팅창을 떠나지 않고** 키워드 평가·콘텐츠 갭 분석·outlier 발굴 가능.
- **자동 팝업으로 연결되며**, VidIQ 브라우저 확장 + 계정만 있으면 추가 설정 거의 불필요.
- 진짜 가치는 **Claude Project + 매주 누적되는 본인 채널 데이터**와 결합할 때. 매주 콘텐츠 회의 2시간 → 15분으로 압축 가능.
