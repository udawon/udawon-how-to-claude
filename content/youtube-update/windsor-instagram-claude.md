## 이게 뭔가요?

**Windsor**는 Instagram·LinkedIn·Facebook·YouTube 같은 마케팅 채널의 분석 데이터를 한곳에 모으는 서비스(SaaS, 구독형 클라우드 서비스)입니다. 이 Windsor가 **Claude의 커넥터(connector)**로 연결되면서, Claude가 내 Instagram 비즈니스 인사이트(insight, 분석 지표)를 직접 읽을 수 있게 되었습니다.

여기서 **MCP Connector(커넥터)**는 "AI에게 외부 서비스의 데이터·기능을 안전하게 연결하는 통로"입니다. 마치 콘센트에 코드를 꽂는 것과 비슷합니다 — Claude라는 본체에 Windsor라는 플러그를 꽂으면 Instagram 데이터가 흘러옵니다.

**예전 방식과 신규 방식의 차이.**

| 항목 | 예전: Chrome 확장 | 신규: Windsor 커넥터 |
|------|------------------|---------------------|
| 데이터 가져오기 | 스크린샷 캡처 | 분석 API(앱끼리 데이터를 주고받는 통로)로 직접 |
| 토큰(AI가 처리하는 텍스트 양) | 폭식 — Pro 플랜이면 세션 금방 소진 | 거의 안 씀 |
| 분석 깊이 | 표면 수치(좋아요·댓글 수) | 리텐션 곡선(영상 어디서 사람들이 떠나는지), 좋아요 발생 시점, CTA 효과 등 |
| 사용 편의 | 매번 새 탭에서 확장 열기 | 채팅 한 번에 연결 |

영상 제작자는 이 방식으로 콘텐츠를 만들기 시작한 지 **2개월 만에 4,000명 → 10만 명**으로 인스타 팔로워를 키웠다고 말합니다.

## 왜 알아야 하나요?

한국에서 인스타·페이스북·유튜브 마케팅을 운영하는 사람에게 가장 큰 고통은 **"왜 이 게시물은 잘 나왔고, 저 게시물은 망했나"**를 모르는 것입니다.

| 표면 데이터로 보이는 것 | 깊은 데이터로 보이는 것 |
|------------------------|------------------------|
| 좋아요 1,200개 | 좋아요가 영상 시작 8초에 80% 발생 → 첫 8초 후크가 핵심 |
| 댓글 50개 | 댓글이 CTA 호출 직후 70% 집중 → 그 CTA 패턴이 효과 |
| 도달 50,000명 | 첫 30초 리텐션 65% → 그 이후 급락 → 길이 조절 필요 |

**누가 써야 하나요?**

- 인스타·페이스북 비즈니스 계정으로 마케팅 운영하는 1인 사업자
- SNS 대행을 하는 마케터·에이전시
- 소상공인 마케팅 담당자
- 본인 채널을 키우려는 인플루언서·크리에이터

영상 제작자가 강조한 한 줄.

> 이 데이터를 손에 쥐면 다음 콘텐츠 대본을 훨씬 더 정확하게 쓸 수 있다.

## 어떻게 하나요?

### 사전 조건 (꼭 확인)

| 항목 | 확인 |
|------|------|
| Instagram 계정 유형 | **비즈니스(business) 또는 크리에이터(creator)** 계정이어야 함. 개인 계정은 인사이트 API 접근 불가 |
| Instagram-Facebook 연결 | Instagram이 Facebook 페이지와 연결되어 있어야 로그인 흐름이 작동 |
| Claude 플랜 | Pro 또는 그 이상. Max 플랜이면 더 안정적 |

**계정 전환 방법.** 인스타 앱 → 프로필 → 메뉴 → 설정 → 계정 → 프로페셔널 계정으로 전환.

### 1단계: Claude에 Windsor 커넥터 추가

1. Claude(claude.ai) 로그인
2. 좌측 메뉴 → **Customize(개인화)** → **Connectors(커넥터)**
3. 우측 상단 "+" 또는 "Browse connectors(커넥터 탐색)" 클릭
4. 검색창에 `Windsor` 입력 → **Windsor AI** 선택 → 추가
5. 권한 요청 팝업이 뜨면 **Allow / 허용** 클릭

### 2단계: Windsor 회원 가입과 Instagram 연결

1. Windsor 사이트로 이동(Claude가 자동으로 안내)
2. 계정 생성 또는 기존 계정 로그인
3. 메인 페이지에서 연결 가능한 채널 목록 확인 (LinkedIn·Facebook 페이지·YouTube·Instagram 등)
4. **Instagram & Sites** 클릭
5. Facebook 로그인 창으로 이동 → 평소 인스타 관리하는 Facebook 계정으로 로그인
6. 권한 화면에서 분석할 Instagram 계정 선택 → Finish
7. Windsor 대시보드에 연결된 계정이 표시되면 완료

### 3단계: Claude 채팅에서 커넥터 활성화 확인

1. Claude로 돌아가 새 채팅(New chat) 시작
2. 입력창 옆 **"+" → connectors(또는 More connectors)** 클릭
3. **Windsor AI**가 토글 on(파란색) 상태인지 확인 → 꺼져 있으면 켜기

### 4단계: 분석 프롬프트 던지기

영상 제작자가 사용한 분석 프롬프트의 한국어 버전.

<div class="example-case">
<strong>실전 분석 프롬프트 (한국어)</strong>

```
Windsor를 사용해 내 Instagram 프로필에 연결한 뒤
지난 30일 데이터로 다음 항목을 포함한 리포트를 작성해줘.

1. 기간 전체 개요
2. 성장 지표: 팔로워 증감, 성장률
3. 청중 인구 통계: 연령·성별·지역
4. 인게이지먼트 성과: 평균 인게이지먼트율, 시계열 변화
5. 주요 콘텐츠: 도달·인게이지먼트 상위 5개 게시물(포맷별)
6. 부진한 콘텐츠: 인게이지먼트 하위 게시물과 공통 패턴
7. 포맷·시간대 분석: 어떤 포맷·시간이 가장 효과적이었나
8. 진단: 무엇이 잘되고, 무엇이 안 되는가
9. 다음 30일 행동 3~5개: 구체적이고 실행 가능한 제안
10. 상위 3개 게시물 기반 다음 영상 대본 초안 3편

마지막에 결과를 HTML 리포트 형식으로도 만들어줘 (다운로드용).
```

영상에서 이 프롬프트를 던졌을 때 Claude가 Windsor → Instagram 순으로 데이터를 끌어와 일자별 노출·인게이지먼트·청중 분석까지 표 형식으로 정리해 주었습니다.

### Claude의 동작 흐름

```
사용자 프롬프트
   ↓
Claude → Windsor 커넥터에 권한 요청
   ↓
Windsor → Instagram 분석 API 호출
   ↓
Instagram 인사이트 데이터를 Windsor가 정리
   ↓
Claude가 받은 데이터로 리포트 작성
   ↓
사용자가 "HTML로 만들어줘" 추가 요청 → 다운로드 가능한 페이지 출력
```

## 실전 예시

<strong>실전 케이스: 1인 약국의 인스타 운영</strong>

상황: 약국 인스타 1,500팔로워. 영양상담·복약지도 카드 뉴스를 주 3회 올리지만, 어떤 게시물이 진짜 효과 있는지 감으로만 판단.

순서.

1. 인스타 계정을 비즈니스 계정으로 전환 (이미 되어 있다면 패스)
2. Claude에 Windsor 커넥터 추가, Windsor에 인스타 연결
3. 첫 분석 프롬프트:
   ```
   지난 60일 동안 내 인스타에서 인게이지먼트 상위 5개 게시물과 하위 5개를 비교해줘.
   각 게시물의 후크(첫 화면 카피), 주제, 형식, 게시 시간을 표로 정리.
   공통 패턴이 있다면 가설로 제시.
   ```
4. Claude가 받은 데이터로 분석 결과 도출 — 예: "복약지도형 카드뉴스가 영양제 추천형보다 평균 인게이지먼트 2.3배"
5. 다음 30일 콘텐츠 캘린더에 복약지도형 비중을 60%로 늘림
6. 한 달 뒤 같은 프롬프트 재실행 → 변화 검증

추가. 결과를 HTML 리포트로 출력해 보호자·동료 약사에게 공유하면 협업 도구로도 활용 가능.

## 주의할 점

**개인 계정은 못 씁니다.** 가장 흔한 실수 — 프로페셔널 계정으로 전환하지 않으면 Windsor가 인사이트 API에 접근할 수 없어 빈 데이터가 옵니다. 시작 전 반드시 확인.

**Instagram-Facebook 연결 끊김 주의.** 인스타가 어떤 Facebook 계정과 연결되어 있는지 모르면 로그인 단계에서 헤맵니다. 영상 제작자도 처음 연결 시 다른 페이스북에 묶인 인스타가 떠서 한 번 변경했습니다. 미리 인스타 → 설정 → 계정 → 페이스북에서 어느 페이지와 연결되었는지 확인하세요.

**응답이 가끔 느립니다.** 영상 제작자(Max 플랜)도 일부 응답이 평소보다 오래 걸린다고 언급. 데이터 양이 많을수록 길어집니다. 한 번에 너무 긴 기간(6개월 이상)을 요청하면 시간이 오래 걸리거나 잘림 — 30~60일 단위로 나눠 분석하세요.

**대규모 채널은 데이터 페이지네이션 한계가 있을 수 있음.** 게시물 수가 매우 많은 채널은 Windsor가 일정 페이지까지만 가져올 수 있습니다. 핵심 기간을 좁혀 요청하는 편이 안전.

**Windsor는 Instagram 외에도 다른 채널 지원.** LinkedIn·Facebook 페이지·YouTube도 같은 방식으로 연결 가능 — 멀티 채널 통합 대시보드로 확장할 여지가 있습니다.

**민감한 비즈니스 데이터 권한.** Windsor에 인스타 연결을 허용하면 Windsor도 그 데이터에 접근하게 됩니다. 클라이언트 계정을 연결할 때는 사전 동의를 받아 두는 편이 분쟁 방지에 좋습니다.

## 정리

- Windsor MCP 커넥터로 Claude가 Instagram 분석 API에 직접 접근 — 토큰 거의 안 쓰면서 리텐션·CTA 효과 같은 깊은 데이터까지 분석 가능.
- 사전 조건 단 두 줄: ① 인스타가 비즈니스/크리에이터 계정 ② Instagram-Facebook 연결 확인.
- 분석 결과를 HTML 리포트로 출력 → 클라이언트 보고서·팀 공유로 즉시 활용.
