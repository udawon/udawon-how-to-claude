## 이게 뭔가요?

Claude Code가 자율적으로 코드를 작성하고 명령어를 실행할 때, **실수로 위험한 일**을 할 수도 있어요. 데이터베이스 폴더를 삭제한다거나, 환경변수를 바꿔버린다거나.

**NAH**(Not Authorized Here)는 이런 위험한 명령을 **실행 전에 가로채서 차단**하는 도구입니다.

> 비유: 신입사원이 열심히 일하는 건 좋은데, 가끔 서버실 전원을 끄려고 할 수 있잖아요. NAH는 **"잠깐, 이거 괜찮은 거 맞아?"라고 물어보는 선임**과 같아요.

## 왜 알아야 하나요?

Claude Code를 자율 모드(autonomous mode)로 돌릴 때의 위험:

- 프로젝트 디렉토리 밖의 파일을 수정하려 함
- `rm -rf` 같은 재귀적 삭제 명령 실행
- 예상치 못한 네트워크 호출
- 환경변수나 시스템 설정 변경

**NAH가 있으면**: 이런 명령이 실행되기 **전에** 가로채서, 차단하거나 확인을 요청합니다.

## 어떻게 하나요?

### NAH의 작동 원리

NAH는 Claude Code와 시스템 사이에 **권한 레이어**로 위치합니다:

```
Claude Code → NAH(차단/허용 판단) → 시스템 실행
```

단순히 금지 명령 목록을 쓰는 게 아니라, **컨텍스트를 인식**합니다:
- 이 파일이 작업 디렉토리 안에 있나?
- 이 명령이 요청한 작업 범위 안에 있나?
- 합리적인 코딩 작업에 필요한 명령인가?

### 설정하기

규칙 파일에 허용/차단 패턴을 정의합니다:

<div class="example-case">
<strong>💬 예시: 기본 규칙 설정</strong>

**허용 목록**:
- 프로젝트 파일 읽기
- `src/` 디렉토리에 쓰기
- `npm run build`, `npm test` 같은 빌드 명령

**차단 목록** (기본 차단 또는 확인 필요):
- 재귀적 삭제 (`rm -rf`)
- 예상치 못한 네트워크 호출
- 시스템 경로 쓰기 (`/etc/`, `C:\Windows\` 등)
- 환경변수 수정

</div>

### 확인 모드

NAH는 **조용히 차단**하는 대신, 에이전트가 뭘 하려는지 보여주고 직접 승인/거부할 수 있습니다:

<div class="example-case">
<strong>💬 예시: 확인 모드 동작</strong>

```
⚠️ NAH: Claude Code가 다음 작업을 요청합니다:
   작업: 파일 삭제
   경로: /home/user/project/database/
   위험도: 높음

   [승인] [거부]
```

→ 주니어 개발자가 "이거 해도 될까요?" 하고 물어보는 것처럼, AI도 중요한 작업 전에 허락을 구합니다.

</div>

## 실전 예시

<div class="example-case">
<strong>📌 실전 케이스: 자율 모드에서 안전하게 작업하기</strong>

**상황**: Claude Code를 자율 모드로 돌려서 밤새 리팩토링을 시킴

**NAH 없이**: 아침에 와보니 `.env` 파일이 사라짐. 환경변수 재설정에 2시간 소요.

**NAH 있으면**:
1. Claude Code가 `.env` 파일 수정을 시도
2. NAH가 감지: "환경변수 파일 변경은 확인이 필요합니다"
3. 해당 작업만 멈추고, 나머지 리팩토링은 계속 진행
4. 아침에 확인 요청 목록만 처리하면 됨

</div>

## 주의할 점

- **보안 샌드박스가 아님**: NAH는 애플리케이션 레벨의 가드. Docker 컨테이너나 VM 같은 OS 레벨 격리가 아닙니다. **안전벨트이지, 롤케이지가 아닙니다.**
- **의도를 이해하지 못함**: 파일 경로와 명령 패턴으로 판단할 뿐, "이 삭제가 논리적으로 맞는지"까지는 모릅니다
- **규칙 설정이 중요**: 너무 느슨하면 위험하고, 너무 엄격하면 작업이 안 됨. 프로젝트에 맞게 조정 필요
- **컨텍스트 인식에도 한계**: 파일 경로, 명령 유형, 범위 추정은 가능하지만, 깊은 의미 분석은 불가

## 정리

- NAH = Claude Code의 위험한 명령을 실행 전에 차단하는 권한 가드
- 컨텍스트 인식 기반으로 허용/차단/확인 요청 판단
- 자율 모드에서도 안심하고 에이전트를 돌릴 수 있는 안전장치
