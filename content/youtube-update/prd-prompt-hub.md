## 이게 뭔가요?

새로운 서비스나 기능을 만들기 전에 **"뭘 만들 건지" 정리한 문서**가 필요하잖아요. 이걸 PRD(Product Requirements Document, 제품 요구사항 문서)라고 불러요. 쉽게 말하면 **"제품 기획서"**입니다.

보통 기획서 하나 만들려면 몇 시간에서 며칠이 걸리는데, **PRD Prompt Hub**를 사용하면 Claude나 ChatGPT한테 체계적으로 질문을 던져서 **10분 만에 기획서 초안**을 완성할 수 있어요.

비유하면 이래요:

> 원래: 빈 종이에 처음부터 기획서를 직접 작성 (레스토랑 메뉴를 백지에서 구성)
> PRD Prompt Hub: 이미 잘 짜인 양식에 내 프로젝트 정보만 채워 넣기 (메뉴판 템플릿에 우리 요리만 넣기)

## 왜 알아야 하나요?

- **기획서 없이 바로 코딩** → 나중에 "이거 왜 만들었지?" 혼란, 방향 수정에 시간 낭비
- **기획서 작성에 너무 오래 걸림** → 정작 만들 시간이 부족
- **Claude Code에 기획서를 넣으면** → 코드 품질이 훨씬 좋아짐 (AI가 맥락을 정확히 이해하니까)

PRD Prompt Hub를 쓰면 "빠르게 + 체계적으로" 기획서를 뽑을 수 있고, 이 기획서를 Claude Code에 바로 넣어서 개발까지 이어갈 수 있어요.

## 어떻게 하나요?

상황에 따라 **4가지 방법** 중 골라 쓰면 됩니다.

### 방법 1: 싱글샷 프롬프트 — "가장 빠른 길"

프롬프트 한 번에 전체 기획서가 나오는 방식이에요. 속도가 생명일 때 쓰세요.

<div class="example-case">
<strong>💬 예시: 싱글샷으로 F1 대시보드 기획서 만들기</strong>

Claude에 아래처럼 붙여넣기만 하면 됩니다:

```
당신은 전문 프로덕트 매니저이자 기술 아키텍트입니다.
다음 프로젝트의 프로덕션 레디 PRD를 작성하세요.

[프로젝트 설명]
F1 팬을 위한 역대 경기 데이터 대시보드.
드라이버 비교, 시즌별 성적, 트랙 기록을 시각화.

[제약 조건]
- Next.js + TypeScript
- 무료 F1 API 활용
- 모바일 반응형

마크다운 형식으로 PRD만 출력하세요.
```

결과: 기능 요구사항, 우선순위, 수용 기준, 스프린트 계획까지 포함된 기획서가 한 번에 나옵니다.

</div>

**팁**: 프로젝트 설명은 완벽할 필요 없어요. 머릿속에 떠오르는 대로 적고, Claude에게 "이걸 깔끔하게 정리해줘"라고 하면 됩니다.

### 방법 2: 스캐폴드 프롬프트 — "검토 후 확정"

2단계로 나눠서 진행하는 방식이에요. 먼저 **개요(목차)**를 받아보고, 확인한 뒤에 전체 문서를 생성해요.

건물 짓기에 비유하면: 설계도 초안(개요) → 검토 → 본 설계(전체 PRD)

<div class="example-case">
<strong>💬 예시: 2단계 기획서 작성</strong>

**1단계** — 개요 생성 프롬프트를 붙여넣으면 각 섹션별 핵심 항목이 불릿 포인트로 나옵니다.

**검토** — "결제 기능 빠졌네? 추가해줘" 같은 피드백을 줍니다.

**2단계** — 수정된 개요를 기반으로 전체 PRD가 생성됩니다.

이렇게 하면 20페이지 기획서를 다 받은 후에 "아, 이거 아닌데..." 하는 상황을 방지할 수 있어요.

</div>

### 방법 3: PRD 위자드 — "질문에 답하면 기획서 완성"

Claude Code CLI에 **슬래시 커맨드**로 설치해서 쓰는 방식이에요. 뭘 적어야 할지 모를 때 특히 좋아요.

설치 방법:

1. GitHub 레포(snapperai/prd-prompt-hub)에서 설치 프롬프트를 복사
2. Claude Code에 붙여넣기 → 자동으로 슬래시 커맨드 설치
3. Claude Code 재시작
4. `/prd-wizard` 입력하면 시작!

<div class="example-case">
<strong>💬 예시: PRD 위자드 사용 과정</strong>

```
> /prd-wizard

Q1. 뭘 만드나요?
→ "F1 데이터 대시보드"

Q2. 누가 쓰나요?
→ "F1 팬"

Q3. 핵심 기능은?
→ "드라이버 비교, 시즌 기록 조회"

... (총 8개 질문)

→ 전체 PRD 자동 생성!
```

카페에서 주문하는 것처럼, 점원(Claude)이 하나씩 물어보고 알아서 만들어주는 느낌이에요.

</div>

**Claude Code vs Codex CLI 차이점:**

| 도구 | PRD 스타일 | 추천 상황 |
|------|-----------|----------|
| Claude Code | 상세하고 풍부한 기획서 | 이해관계자에게 공유할 때 |
| Codex CLI | 짧고 핵심만 담긴 기획서 | 바로 개발에 들어갈 때 |

둘을 섞어 쓸 수도 있어요 — Claude로 상세 PRD를 만들고, Codex로 개발하는 식으로요.

### 방법 4: Claude 프로젝트 킷 — "장기전용 세팅"

Claude 웹에서 **프로젝트**를 만들어 두는 방식이에요. 한번 세팅하면 매번 프롬프트를 복붙할 필요 없이, 대화만 시작하면 기획서 형식이 자동 적용돼요.

<div class="example-case">
<strong>💬 예시: Claude 프로젝트 세팅</strong>

1. Claude 웹 → 좌측 "Projects" → "Create new project"
2. 프로젝트 이름: "PRD 생성기"
3. Description에 GitHub 레포의 설명 복사
4. Instructions에 지침 복사
5. Knowledge에 지식 파일 업로드

세팅 완료 후:
```
"F1 대시보드 만들어줘"
```
→ Claude가 자동으로 기획서 형식에 맞춰 질문하고 PRD 생성
</div>

기획서를 자주 쓰는 사람에게 추천해요. 매번 세팅할 필요 없이 대화만 시작하면 되니까요.

## 실전 예시

<div class="example-case">
<strong>📌 실전 케이스: 비개발자가 앱 기획서 만들기</strong>

**상황**: 사내에서 "재고 관리 앱을 만들자"는 이야기가 나왔는데, 기획서를 써본 적이 없다.

**해결**:
1. PRD 위자드(`/prd-wizard`)를 실행
2. "재고 관리 앱"이라고만 답하고, 나머지는 "너가 결정해"로 진행
3. 8개 질문에 답하니 기능 목록, 우선순위, 일정 계획이 포함된 기획서가 자동 생성
4. 이 기획서를 Claude Code에 넣고 "이 기획서대로 만들어줘"

**결과**: 기획 경험 없이도 체계적인 기획서 완성 → 바로 개발 착수!

</div>

<div class="example-case">
<strong>📌 실전 케이스: 스캐폴드로 팀 공유용 기획서 작성</strong>

**상황**: 새 기능을 기획 중인데, 팀원들에게 공유하기 전에 빠진 게 없는지 확인하고 싶다.

**해결**:
1. 스캐폴드 프롬프트로 **개요 먼저 생성**
2. 개요를 보니 "에러 처리 전략"이 빠져있음 → 추가 요청
3. 수정된 개요로 전체 PRD 생성
4. 마크다운 파일로 다운로드 → 팀 노션에 공유

**결과**: 2단계 검증 덕분에 빈틈 없는 기획서 완성!

</div>

## 4가지 방법 한눈에 비교

| 방법 | 속도 | 품질 통제 | 적합한 사람 |
|------|------|----------|------------|
| 싱글샷 | ⚡ 가장 빠름 | 낮음 | "일단 빨리 초안 필요!" |
| 스캐폴드 | 보통 | 높음 (개요 검토) | "꼼꼼하게 확인하고 싶어" |
| PRD 위자드 | 보통 | 보통 | "뭘 넣어야 할지 모르겠어" |
| Claude 프로젝트 | 초기 세팅 필요 | 높음 (지속적) | "기획서를 자주 써야 해" |

## 주의할 점

- **프로젝트 설명이 구체적일수록** 기획서 품질이 좋아져요. "앱 만들어줘"보다 "F1 팬을 위한 데이터 대시보드, 드라이버 비교 기능 포함"이 훨씬 좋은 결과를 줍니다.
- **타임라인은 참고용**: AI가 생성하는 일정(8주 스프린트 등)은 전통적 개발 기준이에요. AI와 함께 만들면 훨씬 빠르니 일정은 조정하세요.
- **슬래시 커맨드 설치 후**: Claude Code를 반드시 **재시작**해야 새 커맨드가 보여요.
- **기획서 = 출발점**: AI가 만든 기획서도 완벽하진 않아요. 반드시 검토하고 내 상황에 맞게 수정하세요.

## 정리

- PRD Prompt Hub = Claude/ChatGPT로 기획서를 빠르게 만드는 **4가지 템플릿 모음**
- 상황에 따라 싱글샷(속도), 스캐폴드(검토), 위자드(가이드), 프로젝트(장기)를 선택
- GitHub에 공개(MIT 라이선스)되어 있어 누구나 무료로 사용 가능
