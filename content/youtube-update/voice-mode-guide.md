## 이게 뭔가요?

Voice Mode는 **키보드로 타이핑하는 대신 말로 Claude Code에게 지시**하는 기능입니다.

> 비유: 카카오톡에서 메시지 치는 대신 **음성 메시지** 보내는 것과 같아요. 머리속에 정리는 됐는데 타이핑하기 귀찮을 때, 그냥 말하면 됩니다.

```
타이핑: "이 함수 테스트 코드 작성해줘"
Voice Mode: 스페이스바 꾹 누르고 "이 함수 테스트 코드 작성해줘" → 끝
```

결과는 똑같고, 입력 방식만 달라집니다. 한국어, 영어 모두 인식되고 **섞어서 말해도** 괜찮습니다.

## 왜 알아야 하나요?

- **타이핑보다 말이 빠름**: 특히 긴 지시사항을 전달할 때
- **한글/영어 혼용 OK**: "이 파일을 read해서 summary로 만들어 줘" 같은 혼용도 정확히 인식
- **손이 자유로워짐**: 코드를 보면서 음성으로 지시 가능
- **비개발자에게 더 친숙**: 타이핑보다 말하기가 자연스러운 사람에게 진입장벽을 낮춰줌

## 어떻게 하나요?

### 1단계: Claude Code 최신 버전 업데이트

```bash
claude update
```

현재 버전과 최신 버전을 확인하고 업데이트합니다.

### 2단계: Voice Mode 활성화

Claude Code를 실행한 후:

```
/voice
```

처음 실행하면 필요한 패키지 설치 안내가 나옵니다. "안내대로 설치해줘"라고 하면 자동으로 설치됩니다.

설치 후 `Ctrl+C` 두 번으로 재시작하면 Voice Mode가 활성화됩니다.

### 3단계: 사용하기

## 트러블슈팅

### 마이크가 안 잡힐 때

**Mac**: 시스템 설정 → 사운드 → 입력 → 마이크 선택 확인. 입력 레벨이 말할 때마다 올라가는지 확인.

**Windows**: 설정 → 시스템 → 사운드 → 입력에서 확인.

### 텍스트 변환이 이상할 때

Claude Code 설정에서 **Language** 검색 → 한국어로 설정:
1. 설정 → Language 검색
2. 한국어 선택 → 적용 → 엔터

> 처음에 설정 안 하고 쓰다가 한국어를 제대로 인식 못하는 경우가 있어요. 이 설정 하면 확 나아집니다.

### 재시작할 때마다 Voice Mode 꺼짐

현재 Voice Mode는 Claude Code를 껐다 켜면 자동으로 꺼집니다. 매번 `/voice`로 다시 켜야 합니다.

> 이 부분은 조만간 개선될 것으로 보입니다.

## 실전 예시

<div class="example-case">
<strong>📌 실전 케이스: 음성으로 코드 리뷰 요청</strong>

**상황**: 코드를 읽으면서 수정 사항을 지시하고 싶은데, 타이핑하려면 코드에서 눈을 떼야 함

**Voice Mode 활용**:
1. 코드를 눈으로 읽으면서
2. 스페이스바 꾹 → "이 함수에서 에러 핸들링이 빠져있어. try-catch로 감싸고 에러 로그도 추가해줘"
3. 손 떼기 → 확인 → 엔터

**결과**: 코드에서 시선을 떼지 않고 자연스럽게 지시 가능
</div>

## 주의할 점

- **스페이스바는 "꾹 누르고 있기"**: 톡 한 번 누르면 그냥 스페이스가 입력됨. 워키토키처럼 누르고 있어야 함
- **인식률이 완벽하지 않을 수 있음**: AI가 이해하는 데는 문제 없지만, 사람이 보기에 어색한 텍스트가 나올 수 있음
- **재시작 시 재활성화 필요**: `/voice`를 매번 다시 입력해야 함 (추후 개선 예정)
- **조용한 환경 추천**: 주변 소음이 크면 인식률이 떨어질 수 있음

## 정리

- Voice Mode = 스페이스바 꾹 누르고 말하면 Claude Code가 실행
- 한국어, 영어, 혼용 모두 OK
- 설정: `claude update` → `/voice` → 스페이스바 꾹 → 말하기 → 엔터
