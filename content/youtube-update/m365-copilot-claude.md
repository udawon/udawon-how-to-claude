## 이게 뭔가요?

Microsoft 365 Copilot의 **Wave 3 업데이트**로, 이제 Copilot 안에서 **Claude 모델을 선택해서 사용**할 수 있게 되었습니다.

> 비유: 원래 회사 구내식당에 한식만 있었는데, 이제 **양식 셰프도 추가**된 거예요. 같은 식당(M365)에서 메뉴(AI 모델)를 골라서 주문할 수 있는 겁니다.

기존에는 GPT 모델만 사용 가능했지만, 이제 Anthropic의 Claude를 포함한 **멀티모델** 환경이 됐습니다.

## 왜 알아야 하나요?

- **Excel, PowerPoint, Word에서 Claude 활용**: 기존 M365 앱 안에서 바로 사용
- **모델 선택권**: 작업에 따라 GPT와 Claude 중 더 적합한 모델을 골라 쓸 수 있음
- **Copilot Cowork**: 에이전트가 백그라운드에서 작업을 수행하는 새로운 기능
- **기업 환경에서 Claude 활용**: 별도 도구 없이 기존 업무 도구 안에서 AI 활용 가능

## 어떻게 하나요?

### Wave 3의 핵심 변경사항

**1. 멀티모델 지원**

Copilot Chat에서 모델을 선택할 수 있습니다. GPT 계열 외에 Claude를 선택하면 Anthropic 모델이 응답합니다.

<div class="example-case">
<strong>💬 예시: Copilot에서 Claude 사용</strong>

```
Copilot Chat → 모델 선택 드롭다운 → "Claude" 선택
→ Excel 데이터 분석, 문서 요약, 이메일 작성 등에 Claude가 응답
```

작업 특성에 따라 모델을 바꿔가며 사용할 수 있어요.
</div>

**2. Copilot Cowork**

에이전트가 **백그라운드에서 작업**을 수행하는 기능입니다. 지시를 내리고 다른 일을 하다가, 완료되면 알림을 받는 방식.

**3. Agent 365**

커스텀 에이전트를 만들어서 반복 작업을 자동화할 수 있습니다.

### Excel + Claude 활용

<div class="example-case">
<strong>💬 예시: Excel에서 Claude로 데이터 분석</strong>

```
1. Excel 파일 열기
2. Copilot 패널에서 Claude 모델 선택
3. "이 매출 데이터에서 분기별 트렌드를 분석하고, 이상치를 찾아줘"
4. Claude가 분석 결과와 차트 추천을 제공
```

</div>

### PowerPoint + Claude 활용

WBS(작업분류체계)가 담긴 Excel에서 → PowerPoint 전체 스케줄 슬라이드를 자동 생성하는 것도 가능합니다.

## 실전 예시

<div class="example-case">
<strong>📌 실전 케이스: 회의 자료 자동 생성</strong>

**상황**: Excel에 정리된 프로젝트 데이터를 기반으로 경영진 보고용 PowerPoint를 만들어야 함

**활용법**:
1. Excel 데이터를 Copilot(Claude 모델)으로 분석
2. "이 데이터를 기반으로 경영진 보고용 핵심 포인트 5개를 뽑아줘"
3. PowerPoint에서 Copilot으로 슬라이드 자동 생성
4. 디자인 다듬기

**결과**: 2시간 걸리던 보고서 작성이 20분으로 단축

</div>

## 주의할 점

- **M365 E7 라이선스 필요할 수 있음**: Wave 3의 일부 기능은 새로운 E7 라이선스에 포함. 기존 라이선스로 사용 가능한 범위를 확인하세요.
- **Claude 모델 선택은 Copilot Chat에서**: 모든 M365 앱에서 동일하게 사용 가능한 것은 아닐 수 있음
- **기업 정책 확인**: 회사에서 특정 AI 모델 사용을 제한할 수 있으니 IT 부서 확인 필요
- **데이터 보안**: 기업 데이터가 어떤 모델로 처리되는지 보안 정책과 대조 필요

## 정리

- M365 Copilot Wave 3로 Claude 모델을 Copilot 안에서 직접 사용 가능
- Excel, PowerPoint, Word 작업에 Claude의 분석력 활용
- Copilot Cowork로 백그라운드 에이전트 작업도 가능
