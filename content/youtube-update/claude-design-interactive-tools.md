## 이게 뭔가요?

Claude Design(클로드 디자인, Claude의 시각 산출물 생성 기능 — 웹 페이지·인터랙티브 앱·차트 등을 자연어로 만들 수 있는 모드)으로 누구나 자기 분야의 전문 지식을 **퀴즈·추천 도구·인터랙티브 지도**로 바꿀 수 있습니다.

마치 빵집 사장님이 "우리 빵을 좋아할 사람을 찾는 퀴즈"를 직접 디자인하는 것과 비슷합니다. 예전에는 개발자에게 의뢰해 몇 주 걸렸을 결과물이 **프롬프트(prompt, AI에게 보내는 요청 메시지) 한 번**으로 몇 분 안에 나옵니다.

영상 제작자가 만든 6가지 사례를 보고 나면, 자기 분야에 어떻게 적용할지 바로 떠오릅니다.

## 왜 알아야 하나요?

**고객은 "자기 자신에 대해 더 알고 싶어 합니다."** 영상 제작자의 핵심 통찰: 사람은 본능적으로 자기 자신을 이해하려 들고, 인터랙티브 콘텐츠는 그 욕구를 정확히 자극합니다.

이를 활용하면 다음이 가능해집니다.

| 효과 | 설명 |
|------|------|
| 청중 참여 | 글·영상보다 체류 시간이 길다 |
| 데이터 수집 | 응답을 통해 청중이 어떤 사람인지 파악 |
| 리드(lead, 잠재 고객) 확보 | 결과 보기 직전에 이메일 입력 단계 추가 가능 |
| 권위 강화 | "내 분야의 전문가가 만든 추천 도구" 포지셔닝 |
| 추천 후 판매 연계 | 추천 결과 → 본인 상품·서비스로 자연스럽게 연결 |

전문 개발자나 디자이너 외주가 필요 없으므로 **1인 사업자·강사·강의 운영자·마케터**에게 진입 장벽이 가장 낮은 콘텐츠 마케팅 방법 중 하나입니다.

## 어떻게 하나요?

### 핵심 공식

영상 제작자가 6개 도구 모두에 적용한 공식은 단순합니다.

```
후보 데이터 + 분기 질문 + (선택) 스타일 = 인터랙티브 도구
```

| 구성 요소 | 의미 | 예시 |
|-----------|------|------|
| 후보 데이터 | 추천할 항목 목록 | 카메라 모델 20종 / 여행지 30곳 / 영양제 15종 |
| 분기 질문 | 결과를 좁히는 질문 | "예산은?", "용도는?", "숙련도는?" |
| 스타일 | 룩앤필 지정 | "neon retro", "다크 모드", "파스텔" |

이 세 가지를 정리해 Claude Design에 던지면 됩니다.

### 6종 사례 한눈에

| # | 도구 | 핵심 내용 | 비즈니스 활용 포인트 |
|---|------|-----------|----------------------|
| 1 | **AI Native 퀴즈** | 10문항 + 진행 바 + 결과 페이지를 단일 프롬프트("one shot")로 생성. "당신은 상위 몇 %의 AI 사용자인가" 같은 자기 평가형 | 자기 평가 욕구 자극 → 청중 참여 |
| 2 | **Which AI 도구 결정기** | "유튜브 음성 클로닝하고 싶어"라고 입력 → 11labs·Descript·Suno 추천 + 가격 + 링크. 별도 Compare(비교) 탭에서 두 도구 직접 비교 | "어떤 도구를 써야 하나" 검색 트래픽 잡기 |
| 3 | **콘텐츠 크리에이터 장비 추천 퀴즈** | 카메라/팟캐스트 분기 → 예산·용도·숙련도 질문 → "Canon R50" 같은 구체 추천 + 부속품 | 어필리에이트(제품 추천 수수료) 수익 결합 가능 |
| 4 | **여행지 추천 퀴즈** | 휴가 목표·원하는 경험 입력 → "Barcelona" 같은 결과 | 여행사·블로거가 자기 추천 로직 시각화 |
| 5 | **시카고 인터랙티브 지도** | 명소 핀 클릭 시 설명 표시. 룩앤필 빠르게 커스터마이즈 | 지역 사업자·관광 가이드의 콘텐츠 자산 |
| 6 | **Bruno Mars 투어 인터랙티브 지도** | 모든 공연 도시·날짜·공연장 정보를 "neon retro" 스타일 지도에 표시 | 팬 콘텐츠·이벤트 정보 시각화 |

### 한 줄 프롬프트 패턴

영상 제작자가 사용한 패턴은 모두 비슷합니다.

<div class="example-case">
<strong>패턴 1: 퀴즈형</strong>

```
"[주제]에 대한 퀴즈를 만들어줘.
- 질문 10개
- 진행 바(progress bar) 표시
- 결과 페이지에 [상위 % 표시 / 추천 결과 표시]
- [원하는 스타일] 룩앤필"
```

영상에서 "AI Native 퀴즈"는 이 패턴을 거의 그대로 사용해 한 번에 완성되었습니다("pretty much in one shot").
</div>

<div class="example-case">
<strong>패턴 2: 추천기형</strong>

```
"[분야] 추천 도구를 만들어줘.
- 분기 질문: [질문 1, 질문 2, 질문 3]
- 후보 목록과 각 항목의 [가격·링크·강점] 데이터:
  [데이터 붙여넣기]
- 결과 화면에 추천 이유와 가격 표시
- 두 옵션을 직접 비교하는 Compare 탭도 추가"
```
</div>

<div class="example-case">
<strong>패턴 3: 인터랙티브 지도</strong>

```
"[지역] 인터랙티브 지도를 만들어줘.
- 핀 위치 데이터: [장소명, 위도, 경도, 설명]
- 핀 클릭 시 설명 팝업 표시
- 스타일: [neon retro / 다크 모드 / 파스텔]"
```

영상의 시카고 지도와 Bruno Mars 투어 지도가 이 패턴입니다.
</div>

## 실전 예시

<div class="example-case">
<strong>실전 케이스: 약사가 운영하는 영양제 추천 퀴즈</strong>

상황: 약국 SNS에 "어떤 영양제를 먹어야 할까요?" 질문이 매주 수십 건. 답하느라 시간이 사라지고, 같은 답을 반복.

순서.

1. 후보 데이터 정리(엑셀이나 표에 영양제 20종 — 비타민D·오메가3·프로바이오틱스 등 — 적응증·주의사항·복용법·가격대)
2. 분기 질문 정리:
   - 주된 고민은? (피로 / 면역력 / 소화 / 수면)
   - 연령대는?
   - 복용 중인 처방약이 있나요?
   - 알레르기 이력은?
3. Claude Design에 "위 후보·질문으로 영양제 추천 퀴즈를 만들어줘. 결과 페이지에 추천 이유와 약사 상담 예약 버튼 포함. 파스텔 그린 톤" 프롬프트
4. 완성 → 약국 인스타 프로필 링크에 연결
5. 결과 페이지 마지막에 "정확한 추천을 원하면 약사 1:1 상담 예약" CTA → 리드 확보

추가 활용. 결과 직전에 이메일 입력 단계를 넣어 "한 달 뒤 영양제 가이드 PDF 보내드립니다" 식으로 이메일 리스트 구축 가능.

</div>

<div class="example-case">
<strong>실전 케이스: 1인 강사의 "어떤 강의가 맞을까" 추천기</strong>

상황: 마케팅 강의를 5종 운영. 신청자에게 "어떤 강의를 들어야 하나요?" 문의가 끊이지 않음.

순서.

1. 5개 강의 데이터 정리 — 대상·난이도·소요 시간·가격·핵심 산출물
2. 분기 질문 4개:
   - 현재 마케팅 경험은? (없음 / 1년 미만 / 1~3년 / 3년 이상)
   - 가장 시급한 문제는? (리드 부족 / 전환율 / 운영 부담)
   - 주당 학습 시간은?
   - 1인 사업자 / 팀장 / 직원
3. Claude Design 프롬프트: "위 데이터·질문으로 강의 추천 퀴즈. 결과는 가장 적합한 강의 1개 + 보조 강의 1개. 결과 페이지에 신청 링크와 가격 표시"
4. 완성 → 홈페이지 메인에 임베드(embed, 외부 페이지에 끼워 넣기)
5. 응답 데이터를 모아 어떤 강의 수요가 가장 많은지 다음 분기 콘텐츠 기획 자료로 활용

영상의 "Which AI" 도구가 정확히 이 구조입니다.

</div>

## 주의할 점

**후보 데이터의 질이 결과 품질을 결정합니다.** Claude는 마법사가 아닙니다. 후보 정보가 부정확하거나 빠져 있으면 추천도 빈약해집니다. **자기 분야의 전문 지식이 결국 핵심 자산.**

**스타일 한 단어가 인상을 결정합니다.** "neon retro", "minimal", "pastel", "다크 모드" 같은 키워드 한두 개로 결과물의 인상이 크게 갈립니다. 영상 제작자도 모든 도구에 스타일 키워드를 넣었습니다.

**한 번에 완성을 노리지 말고 반복 개선.** 영상에서 "Bruno Mars 투어 지도"는 제작자도 "고칠 점이 있다"고 인정했습니다. Claude Design은 결과물을 보고 "여기 색을 바꿔줘", "이 부분에 호버 효과 추가" 식으로 계속 다듬을 수 있습니다.

**리드 수집·결제 연결은 별도 도구 필요.** Claude Design 산출물 자체는 인터랙티브 페이지지만, 이메일 수집이나 결제는 별도 폼·결제 도구(Typeform·Stripe·Toss 등)와 연결해야 실제 비즈니스 자산이 됩니다.

**한국어 데이터·질문은 한국어로.** 영상 사례는 모두 영어이지만, 후보 데이터·질문을 한국어로 던지면 결과도 한국어로 나옵니다. 굳이 영어로 변환할 필요 없습니다.

## 정리

- 자기 분야의 후보 데이터 + 분기 질문 + 스타일 키워드 = 몇 분 만에 인터랙티브 도구 완성.
- 6가지 사례(퀴즈·추천기·여행 추천·인터랙티브 지도)는 거의 모든 업종이 자기 식으로 응용 가능한 공통 골격.
- 결과 페이지에 "상담 예약·이메일 수집·구매 링크"를 붙이면 콘텐츠 자산이 곧바로 리드 확보 자산이 된다.
