# 데스크탑 앱 빠른 시작 가이드

Claude Code 데스크탑 앱을 설치하고 5분 안에 첫 번째 코드 편집을 시작하는 방법을 안내합니다. 터미널이나 명령어를 몰라도 됩니다.

## 예시 케이스

> 콘텐츠 마케터 최다영 씨는 회사 블로그의 WordPress 플러그인 코드에 오류가 생겼습니다. 데스크탑 앱을 설치하고 플러그인 폴더를 선택한 뒤 "함수가 실행되지 않는 오류를 찾아서 고쳐줘"라고 입력합니다. Claude가 변경사항을 보여주면 "승인"을 클릭하는 것으로 끝입니다. 코드를 전혀 몰라도 됩니다.

---

## 앱 구성 탭 소개

데스크탑 앱에는 3개의 탭이 있습니다:

| 탭 | 설명 |
|----|------|
| **Chat** | 파일 접근 없이 일반 대화 (claude.ai와 동일) |
| **Cowork** | 클라우드 VM에서 자율적으로 백그라운드 작업 수행 |
| **Code** | 내 컴퓨터 파일에 직접 접근하는 코딩 도우미 |

이 가이드는 **Code 탭** 사용에 초점을 맞춥니다.

> Claude Code를 사용하려면 **Pro, Max, Teams, Enterprise 구독**이 필요합니다.

---

## 설치 방법

### 1단계: 앱 다운로드

| 운영체제 | 다운로드 |
|----------|----------|
| macOS | [Universal 버전 다운로드](https://claude.ai/api/desktop/darwin/universal/dmg/latest/redirect) (Intel + Apple Silicon 모두 지원) |
| Windows | [64비트 버전 다운로드](https://claude.ai/api/desktop/win32/x64/exe/latest/redirect) |
| Windows ARM64 | [ARM64 버전 다운로드](https://claude.ai/api/desktop/win32/arm64/exe/latest/redirect) |

> Linux는 현재 지원하지 않습니다.

### 2단계: 로그인

앱을 실행하고 Anthropic 계정으로 로그인합니다.
- macOS: 응용 프로그램 폴더에서 Claude 실행
- Windows: 시작 메뉴에서 Claude 실행

### 3단계: Code 탭 열기

상단 중앙의 **Code** 탭을 클릭합니다.
- 업그레이드 안내가 나오면: 유료 구독 필요
- 온라인 로그인 안내가 나오면: 로그인 완료 후 앱 재시작
- 403 오류가 나오면: 인증 문제 해결 가이드 참조

> 데스크탑 앱에는 Claude Code가 내장되어 있어 Node.js나 CLI를 별도 설치할 필요가 없습니다.

## 첫 번째 세션 시작하기

### 1단계: 환경과 폴더 선택

**Local**을 선택하면 내 컴퓨터에서 직접 실행됩니다.

**폴더 선택**을 클릭해서 작업할 프로젝트 폴더를 선택합니다.

> 처음에는 잘 알고 있는 작은 프로젝트로 시작하는 것을 권장합니다. Windows 사용자는 [Git for Windows](https://git-scm.com/downloads/win) 설치가 필요합니다.

다른 환경 옵션:
- **Remote**: 앱을 닫아도 계속 실행되는 Anthropic 클라우드
- **SSH**: 원격 서버에 연결 (Claude Code가 서버에 설치되어 있어야 함)

### 2단계: 모델 선택

전송 버튼 옆 드롭다운에서 AI 모델을 선택합니다. 세션 시작 후에는 변경할 수 없습니다.

### 3단계: Claude에게 작업 요청하기

자연어로 원하는 작업을 입력합니다. 예를 들어:
- `TODO 주석을 찾아서 수정해줘`
- `메인 함수에 대한 테스트를 추가해줘`
- `이 코드베이스의 사용 방법을 CLAUDE.md로 작성해줘`

### 4단계: 변경사항 검토하고 승인하기

기본 모드인 **승인 요청 모드**에서 Claude는:
1. 각 파일의 변경사항을 비교 화면으로 보여줍니다.
2. **승인** 또는 **거부** 버튼이 나타납니다.
3. Claude가 작업하는 동안 실시간 업데이트를 보여줍니다.

거부하면 Claude가 다른 방법을 물어봅니다. **승인 전까지 파일은 수정되지 않습니다.**

## 시작 후 활용 방법

### 작업 중간에 방향 수정하기
Claude가 잘못된 방향으로 가고 있다면 정지 버튼을 클릭하거나, 수정 내용을 입력하고 Enter를 누르세요. 기다리거나 처음부터 다시 시작할 필요가 없습니다.

### 파일과 이미지 첨부하기
`@파일명`을 입력하거나 첨부 버튼으로 이미지, PDF를 추가합니다. 버그 스크린샷이나 디자인 시안을 붙여넣으면 Claude가 바로 이해합니다.

### 반복 작업을 슬래시 명령으로 저장하기
`/`를 입력하거나 **+** → **슬래시 명령**을 클릭하면 자주 사용하는 명령을 저장하고 재사용할 수 있습니다.

### 자동화 수준 조정하기
처음에는 **승인 요청 모드**로 시작하고, 익숙해지면 **편집 자동 승인 모드**로 전환해 더 빠르게 작업할 수 있습니다. **계획 모드**는 큰 작업 전에 방향을 미리 검토할 때 유용합니다.

### 여러 작업 동시 진행하기
사이드바의 **+ 새 세션**으로 병렬 세션을 열면 여러 작업을 동시에 진행할 수 있습니다. 각 세션은 독립적인 Git 작업 공간을 가집니다.

### PR 자동 모니터링하기
풀 리퀘스트를 생성하면 CI 테스트 결과를 자동으로 모니터링하고, 실패 시 자동 수정도 할 수 있습니다.

### 예약 작업 설정하기
매일 아침 코드 리뷰, 주간 의존성 감사 등을 자동으로 실행하도록 일정을 설정할 수 있습니다.

## CLI 사용자를 위한 안내

데스크탑 앱은 CLI와 동일한 엔진을 사용합니다. 같은 프로젝트에서 동시에 사용할 수 있으며, CLAUDE.md, MCP 서버, 훅, 스킬, 설정 파일을 공유합니다.

CLI 세션을 데스크탑으로 이동하려면 터미널에서 `/desktop`을 입력하세요 (macOS, Windows만 지원).
