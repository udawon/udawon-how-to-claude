# Claude Code 개요

> Claude Code는 여러분의 코드베이스를 읽고, 파일을 수정하고, 명령어를 실행하며, 개발 도구와 통합되는 AI 코딩 도우미입니다.

---

## Claude Code가 뭔가요?

Claude Code를 **경험 많은 개발자 동료**라고 생각해 보세요. 이 동료는 여러분의 프로젝트 전체를 한눈에 파악하고, 여러분이 자연어로 요청하면 직접 코드를 작성하거나 수정하고, 명령어를 실행하고, 버그를 찾아 고쳐줍니다.

단순히 "코드를 보여주는" 도구가 아니라, **실제로 작업을 수행**합니다.

---

## 어디서 사용할 수 있나요?

| 환경 | 특징 | 시작 방법 |
|------|------|-----------|
| **터미널 (CLI)** | 가장 강력한 기능 제공. 파일 편집, 명령어 실행, 전체 프로젝트 관리 | `curl` 명령어로 설치 후 `claude` 입력 |
| **VS Code** | 에디터 안에서 바로 사용. 변경사항 미리보기(diff) 지원 | VS Code 확장 프로그램 설치 |
| **데스크탑 앱** | 여러 세션 동시 관리, 시각적 diff 검토, 예약 작업 | 공식 사이트에서 다운로드 |
| **웹 브라우저** | 설치 없이 브라우저에서 바로 사용 | claude.ai/code 접속 |
| **JetBrains IDE** | IntelliJ, PyCharm, WebStorm 등 지원 | 플러그인 설치 |

> 모든 환경에서 동일한 Claude Code 엔진이 동작합니다. CLAUDE.md 설정, 단축키, MCP 서버 등이 모든 환경에서 공유됩니다.

## 무엇을 할 수 있나요?

### 귀찮은 반복 작업 자동화
테스트 작성, 린트 오류 수정, 충돌 해결, 의존성 업데이트, 릴리즈 노트 작성 등 매일 미루고 싶은 작업들을 대신 처리합니다.

```
claude "auth 모듈에 테스트를 작성하고 실행한 뒤 실패하는 것들을 고쳐줘"
```

### 기능 개발 및 버그 수정
자연어로 원하는 것을 설명하면 Claude Code가 여러 파일에 걸쳐 코드를 작성하고 동작을 확인합니다. 에러 메시지를 붙여넣으면 원인을 추적해 수정합니다.

### Git 작업 자동화
변경사항 스테이징, 커밋 메시지 작성, 브랜치 생성, PR 열기까지 자동으로 처리합니다.

```
claude "내 변경사항을 설명이 잘 된 메시지로 커밋해줘"
```

### MCP로 외부 도구 연결
MCP(Model Context Protocol)는 **블루투스처럼** Claude Code를 외부 서비스에 연결해줍니다. Google Drive 문서 읽기, Jira 티켓 업데이트, Slack 데이터 가져오기 등이 가능합니다.

### CLAUDE.md로 나만의 규칙 저장
CLAUDE.md는 **프로젝트 헌법** 같은 파일입니다. 코딩 기준, 아키텍처 결정, 선호 라이브러리 등을 적어두면 매 세션마다 Claude가 자동으로 읽습니다.

### 여러 Claude 동시 실행 (에이전트 팀)
리드 에이전트가 작업을 나누고, 여러 Claude가 동시에 다른 파트를 작업합니다. 큰 프로젝트를 빠르게 처리할 때 유용합니다.

### 스크립트/파이프라인에 통합
Unix 철학을 따라 다른 도구와 쉽게 연결됩니다.

```bash
# 로그 모니터링 후 이상 감지 시 Slack 알림
tail -f app.log | claude -p "이상한 것이 보이면 Slack으로 알려줘"

# CI에서 자동 번역 PR 생성
claude -p "새 문자열을 프랑스어로 번역하고 검토용 PR을 올려줘"
```

## 예시 케이스

**케이스 1: 쇼핑몰 개발자의 하루**

쇼핑몰 사이트를 개발 중인 민수 씨는 오늘 할 일이 산더미입니다. 결제 모듈 버그, 테스트 누락, 문서 업데이트까지...

```
claude "결제 모듈에서 카드 만료 오류가 발생해. src/payments/ 폴더를 확인해줘"
```

Claude가 관련 파일을 모두 읽고 원인을 찾아 수정안을 제시합니다. 승인하면 직접 코드를 고치고 테스트까지 실행합니다.

**케이스 2: 새 팀원의 온보딩**

큰 회사에 입사한 지영 씨는 수천 개 파일로 이루어진 코드베이스가 어렵습니다.

```
이 프로젝트가 어떻게 동작하는지 설명해줘
인증은 어떻게 처리해?
```

Claude가 마치 선임 개발자처럼 구조와 흐름을 설명해줍니다.

| Slack 버그 리포트 → PR 자동 생성 | Slack 통합 |
| 라이브 웹 앱 디버깅 | Chrome 확장 |
