## 이게 뭔가요?

Claude에게 뭔가를 물어보면 원래 **글자로만** 대답해줬잖아요. 이제는 프롬프트 끝에 **"Show Me"**라고 한마디만 붙이면, 차트, 카드, 애니메이션, 시뮬레이터 같은 **인터랙티브 시각화**를 바로 만들어줍니다.

비유하자면:

> 원래: "이 데이터 분석해줘" → 글자로 줄줄 설명
> Show Me: "이 데이터 분석해줘 Show Me" → 차트와 그래프로 한눈에 보여줌

파워포인트를 열 필요도, 파이썬으로 차트를 그릴 필요도 없습니다. **대화창 안에서 바로** 결과물이 나옵니다.

## 왜 알아야 하나요?

지금까지 Claude한테 복잡한 걸 물어보면, 숫자와 텍스트로 길게 설명해줬습니다. 읽어보면 맞는 말인데... 한눈에 안 들어오죠.

Show Me를 쓰면:
- 투자 수익률 → **인터랙티브 차트**로 슬라이더 움직이면서 확인
- 커리어 옵션 → **카드 형태**로 한눈에 비교
- 복잡한 과정 → **애니메이션 타임라인**으로 이해

**모든 요금제**(월 $20 포함)에서 사용 가능합니다.

## 어떻게 하나요?

### 방법: 프롬프트 끝에 "Show Me" 붙이기

정말 이게 전부입니다. 평소처럼 질문을 쓰고, 맨 끝에 **Show Me**만 추가하면 됩니다.

```
주식 포트폴리오 수익률을 확인하고 싶어.
매월 50만 원을 S&P 500에 투자한다 했을 때
30년에 걸쳐 총 얼마까지 불어날 수 있는지 예측해줘. Show Me
```

<div class="example-case">
<strong>💬 예시: 투자 수익률 시뮬레이션</strong>

**입력:**
```
매월 500만 원을 S&P 500에 30년간 투자하면 얼마가 되는지 예측해줘. Show Me
```

**결과:**
- 투자 원금 18억 → 30년 후 예상 96억
- 수익률 슬라이더를 움직이면 차트가 **실시간으로** 변함
- 보수적(7%) vs 평균(10%) 시나리오 비교 가능
- 투자 기간도 슬라이더로 조절 가능

별도 엑셀이나 계산기 없이 **대화창에서 바로** 시뮬레이션!

</div>

<div class="example-case">

**입력:**
```
집을 구매할 때 30년에 걸쳐 총 얼마를 지불하게 되는지. Show Me
```

**결과에서 직접 입력:**
- 집값: 15.1억
- 기약금: 20%
- 연금리: 5%
- 대출 기간: 30년

→ 월 납입금 648만 원, 총 이자 11.3억, 30년간 총 26.4억 (집값의 1.75배)

### 팁: 결과가 느릴 때

Claude가 시각화를 만드는 데 시간이 걸릴 수 있습니다. 이럴 때는 **대문자로 강조**하면 더 빠르게 반응합니다:

```
DNA가 복제되는 과정을 SHOW ME
```

> Claude는 "조급함을 감지"하기 때문에, 좀 더 적극적으로 요청하면 더 빨리 결과를 만들어냅니다.

### 시각화 안에서 후속 질문하기

Show Me로 만들어진 시각화에는 **클릭 가능한 버튼**이 포함될 수 있습니다. 버튼을 누르면 바로 후속 질문으로 연결되어 대화를 이어갈 수 있습니다.

```
바이브 코딩을 좋아하는 사람이 할 수 있는 일. Show Me
```

→ 카드 형태로 옵션이 나오고 (SaaS 부트스트래퍼, AI 에이전시, 테크 유튜버 등), 각 카드를 클릭하면 **로드맵 질문으로 바로 연결**

## 실전 예시

<div class="example-case">
<strong>📌 실전 케이스: 복잡한 과학 개념 이해하기</strong>

**상황**: DNA 복제 과정이 교과서로는 이해가 안 된다.

**해결:**
```
DNA가 언와인드되면서 복제하는 과정을 SHOW ME
```

**결과**: 헬리케이스 → DNA 풀림 → 폴리머레이스 → 자동 완성까지 **애니메이션으로 재생**되는 시각화 생성. 교과서 10페이지 분량을 한 화면에서 이해 가능.

</div>

<div class="example-case">
<strong>📌 실전 케이스: 도시 계획 — 그림자 시뮬레이터</strong>

**상황**: 우리 동네 빈 공터에 건물을 지으면 그림자가 어떻게 될까?

**해결:**
```
하루 동안 그 공터에 그림자가 어떻게 이동하는지
보여주는 도구를 만들어줘. Show Me
```

**결과**:
- 태양 각도를 슬라이더로 조절 가능
- 겨울(동지), 봄(춘분), 여름(하지), 가을(추분)별 그림자 변화 확인
- 시간대별 그림자 방향 예측

→ 이 시뮬레이터가 실제로 주민 340명의 의견 수렴 → 시청의 공원 개발 승인까지 이어진 사례!

</div>

**결과**: 빈 공터 그림자 조사 → 주민 의견 수렴 → 시청 승인까지의 전체 과정이 **애니메이션 타임라인**으로 생성. Claude Code 긴 세션의 요약에도 활용 가능.

## 주의할 점

- **Show Me는 claude.ai(웹/앱)에서 사용**하는 기능입니다. Claude Code(터미널 CLI)와는 별개
- 복잡한 시각화일수록 생성 시간이 오래 걸릴 수 있음 — 대문자 "SHOW ME"로 강조하면 더 빠름
- 시각화 결과는 대화 안에서만 볼 수 있고, 별도 파일로 저장하려면 스크린샷이 필요

- 프롬프트 끝에 **"Show Me"** 한마디만 붙이면 인터랙티브 시각화 생성
- 차트, 카드, 애니메이션, 시뮬레이터 등 다양한 형태 지원
- 모든 요금제(월 $20 포함)에서 사용 가능 — 별도 설정 없음
