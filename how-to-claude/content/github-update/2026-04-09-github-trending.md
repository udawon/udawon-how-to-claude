## 이번 주 트렌드 한눈에 보기

이번 주는 마치 스마트폰이 처음 나왔을 때처럼, AI를 혼자 쓰는 도구에서 팀으로 굴리는 시스템으로 바뀌는 움직임이 폭발적으로 일어났습니다. Claude Code 관련 도구만 세 개가 동시에 Top 10에 진입했고, 구글은 인터넷 없이 내 기기에서 직접 실행되는 AI 관련 레포를 세 개나 올렸습니다. AI 에이전트(스스로 판단하고 행동하는 AI 프로그램)가 서버에서 내 손 안으로 내려오는 한 주였습니다.

## 1위. oh-my-codex

| 항목 | 내용 |
|------|------|
| 만든 곳 | Yeachan-Heo |
| 주 언어 | TypeScript |
| 이번 주 ⭐ | +11,503 |
| 전체 ⭐ | 19,323 |
| 링크 | [Yeachan-Heo/oh-my-codex](https://github.com/Yeachan-Heo/oh-my-codex) |

### 이게 뭔가요?

OpenAI Codex CLI(오픈AI가 만든 터미널용 AI 코딩 도우미)는 혼자 일하는 어시스턴트입니다. 유능하긴 한데, 혼자라서 복잡한 일은 버겁죠. oh-my-codex는 그 위에 "지휘 본부"를 얹어주는 도구입니다. 마치 혼자 일하던 직원에게 팀장을 붙이고, 팀원들을 배치하고, 업무 일지까지 자동으로 쓰게 만드는 것과 같습니다. Codex 자체는 건드리지 않고, 그 위에 역할 분담·병렬 작업·진행 상황 기록을 추가해줍니다.

### 왜 이번 주에 주목받았나요?

OpenAI가 Codex CLI를 새로 발표하자마자, 가장 빠르게 그 위에 생태계를 얹은 프로젝트가 이것입니다. 출시 직후 단 며칠 만에 19,000개 이상의 스타(좋아요 같은 관심 표시)를 받았는데, "Codex를 고치지 않고 확장한다"는 구조의 참신함이 결정적이었습니다. AI 코딩 도구를 팀 단위로 운영하고 싶은 개발자들의 수요를 정확히 건드렸습니다.

### 핵심 기능

- **역할 기반 워크플로우**: `$deep-interview`, `$ralplan`, `$team` 같은 명령어로 AI에게 "기획자 역할", "검토자 역할"을 부여해 단계별로 작업을 진행시킬 수 있습니다
- **병렬 에이전트 실행**: tmux(터미널 화면 분할 도구)와 worktree(코드 작업 공간 분리 기능)를 이용해 여러 AI 에이전트가 동시에 서로 다른 작업을 처리합니다. 여러 직원이 각자 책상에서 동시에 일하는 것과 같습니다
- **작업 상태 자동 저장**: `.omx/` 폴더에 프로젝트 계획, 작업 로그, 진행 상태를 자동으로 기록합니다. AI가 중간에 멈춰도 어디까지 했는지 기억합니다

<div class="example-case">
<strong>실전 예시: 스타트업 개발자의 밤샘 코드 리뷰</strong>

1인 개발자 이준혁 씨는 매번 혼자 코드를 짜고, 혼자 검토하고, 혼자 문서를 씁니다. 밤새 작업해도 실수가 반복되는 게 고민이었습니다. oh-my-codex를 도입한 뒤 `$team` 워크플로우를 실행하자, AI 에이전트 하나가 코드를 작성하는 동안 다른 에이전트가 동시에 문서를 정리하고 버그 가능성을 검토했습니다. 혼자 4시간 걸리던 작업이 1시간 30분으로 줄었습니다.

</div>

<div class="example-case">
<strong>이런 분께 추천해요</strong>

- **혼자 개발하는 1인 창업자**: AI 한 명으로 부족할 때, 여러 AI를 팀처럼 굴릴 수 있어 실질적인 생산성 향상이 됩니다
- **Codex CLI를 이미 쓰고 있는 개발자**: 기존 환경을 바꾸지 않고 바로 얹을 수 있어 진입 장벽이 없습니다
- **복잡한 프로젝트를 관리하는 팀장급 개발자**: 역할 분담과 진행 상황 기록이 자동화되어 팀 전체 흐름을 잡는 데 도움이 됩니다

</div>

## 2위. openscreen

| 항목 | 내용 |

| 링크 | [siddharthvaddem/openscreen](https://github.com/siddharthvaddem/openscreen) |

### 이게 뭔가요?

Screen Studio는 개발자나 유튜버들이 제품 데모 영상을 만들 때 즐겨 쓰는 유료 앱인데, 월 약 3만 원을 내야 합니다. openscreen은 그것과 똑같은 기능을 무료로, 워터마크(무료 버전에 강제로 붙는 로고) 없이 제공하는 오픈소스(누구나 무료로 쓸 수 있는 공개 소프트웨어) 대안입니다. 비유하자면, 비싼 전문 영상 편집 스튜디오를 그대로 복제해서 누구나 공짜로 쓸 수 있게 만든 것입니다.

### 왜 이번 주에 주목받았나요?

이번 목록 전체에서 스타 수가 가장 많습니다(26,400개 이상). "왜 비싼 돈을 내야 하나"라는 공감대를 정면으로 공략했고, 실제로 써보니 상용 수준과 별 차이가 없다는 후기가 소셜 미디어에 퍼지면서 입소문이 폭발했습니다.

### 핵심 기능

- **자동/수동 줌 + 모션 블러**: 화면을 녹화하면서 클릭한 곳을 자동으로 확대하고, 움직임에 자연스러운 흐림 효과를 넣어줍니다. 전문 영상 편집자가 후보정을 해준 것처럼 보입니다
- **오디오 통합 녹음 + 구간별 속도 조절**: 마이크 목소리와 컴퓨터 소리를 동시에 담고, 지루한 구간은 빠르게, 중요한 부분은 느리게 설정할 수 있습니다
- **텍스트·화살표·이미지 주석**: 영상 위에 설명 글자, 화살표, 이미지를 올려 "여기를 보세요"를 시각적으로 표현합니다. 별도 편집 프로그램 없이 한 번에 완성됩니다

<div class="example-case">
<strong>실전 예시: SaaS 스타트업 세일즈 팀의 데모 영상 제작</strong>

영업 담당 김민지 씨는 고객사에 제품을 소개할 때마다 화면 녹화 후 편집 팀에 맡겨 3~4일을 기다려야 했습니다. openscreen을 사용한 뒤로는 직접 화면을 녹화하면서 클릭할 때마다 자동으로 줌인이 들어가고, 중요한 버튼 옆에 화살표 주석을 달아 30분 만에 완성본을 뽑아냅니다. 고객 미팅 당일 아침에 맞춤형 데모 영상을 만들어 보내는 것이 가능해졌습니다.

</div>

<div class="example-case">
<strong>이런 분께 추천해요</strong>

- **유튜버·콘텐츠 크리에이터**: 비싼 캡처 소프트웨어 구독료를 아끼면서도 전문가 수준의 화면 녹화 영상을 만들 수 있습니다
- **스타트업 세일즈·마케터**: 제품 데모 영상을 편집팀 없이 혼자 빠르게 뽑아낼 수 있어 영업 속도가 빨라집니다
- **온라인 강사·강의 제작자**: 수업 화면을 녹화하면서 중요한 부분에 주석을 달고, 지루한 구간을 자르는 모든 작업을 한 도구에서 처리할 수 있습니다

| 전체 ⭐ | 19,630 |
| 링크 | [google-ai-edge/gallery](https://github.com/google-ai-edge/gallery) |

### 이게 뭔가요?

ChatGPT나 Gemini를 사용하려면 인터넷이 필요하고, 내가 입력한 내용이 서버로 전송됩니다. gallery는 반대입니다. AI 모델을 내 스마트폰 안에 직접 내려받아서, 인터넷 없이도 AI와 대화하고, 사진을 분석하고, 음성을 받아 적을 수 있습니다. 마치 AI를 클라우드(인터넷 서버)가 아닌 내 주머니 속에 넣어두는 것입니다. 구글이 공식으로 만든 체험용 앱입니다.

### 왜 이번 주에 주목받았나요?

구글의 최신 AI 모델인 Gemma 4 출시와 동시에 공개되어 타이밍이 완벽했습니다. "AI를 오프라인으로 쓸 수 있다"는 것 자체가 개인정보 보호 측면에서 큰 의미를 가지는데, 구글이라는 신뢰도 높은 곳에서 직접 앱을 공개했다는 점이 19,600개 이상의 스타로 이어졌습니다.

### 핵심 기능

- **완전 오프라인 AI 채팅 + 추론 과정 시각화**: 인터넷 없이 AI와 대화할 수 있고, AI가 어떤 순서로 생각하는지 단계별로 화면에 보여줍니다
- **사진 분석 + 실시간 음성 받아쓰기 및 번역**: 카메라로 찍은 사진이나 갤러리 이미지를 AI가 분석하고, 말하는 내용을 실시간으로 텍스트로 변환하거나 다른 언어로 번역합니다
- **성능 벤치마크 + 도구 연동 에이전트**: 내 폰에서 AI가 얼마나 빠른지 측정할 수 있고, Wikipedia나 지도 같은 외부 도구와 연결해 정보를 검색하도록 시킬 수 있습니다

<div class="example-case">
<strong>실전 예시: 해외 출장 중 인터넷이 끊긴 상황</strong>

해외 영업 담당 박서준 씨는 현지 공장 방문 중 와이파이도 데이터도 터지지 않는 구역에서 현지어 설명서를 번역해야 하는 상황에 처했습니다. gallery 앱을 미리 설치해둔 덕분에, 스마트폰 카메라로 문서를 찍자 오프라인 AI가 즉시 내용을 분석하고 한국어로 요약해줬습니다. 인터넷 연결 없이, 입력 내용이 외부로 나가지 않은 채로 작업이 완료됐습니다.

</div>

<div class="example-case">
<strong>이런 분께 추천해요</strong>

- **개인정보에 민감한 전문직(의사, 변호사, 회계사)**: 민감한 문서나 대화를 외부 서버에 보내지 않고 AI의 도움을 받을 수 있습니다
- **해외 출장이 잦은 직장인**: 인터넷이 불안정한 환경에서도 번역·분석 기능을 안정적으로 사용할 수 있습니다
- **AI 기술에 관심 있는 일반인**: 복잡한 설정 없이 최신 AI 모델을 스마트폰에서 직접 체험해볼 수 있는 공식 앱입니다

</div>

## 4위. timesfm

| 항목 | 내용 |
|------|------|
| 만든 곳 | google-research |
| 주 언어 | Python (PyTorch, JAX) |
| 이번 주 ⭐ | +3,963 |
| 전체 ⭐ | 15,751 |
| 링크 | [google-research/timesfm](https://github.com/google-research/timesfm) |

### 이게 뭔가요?

일반적인 AI 예측 모델은 "내 데이터로 학습"시키는 과정이 필요합니다. 마치 새 직원을 뽑으면 우리 회사 업무를 처음부터 가르쳐야 하는 것처럼요. timesfm은 이미 구글이 엄청난 양의 시계열 데이터(시간 순서대로 나열된 수치 데이터, 예: 매일의 매출, 시간별 기온)로 학습을 마친 AI입니다. 내 데이터를 가져다 꽂기만 해도 "다음 달 매출이 얼마일지", "이번 주 재고가 얼마나 필요할지"를 바로 예측해줍니다. 새 직원인데 처음부터 전문가인 상태로 오는 것입니다.

### 왜 이번 주에 주목받았나요?

세계 최고 권위의 머신러닝(기계학습) 학회인 ICML 2024에서 발표된 구글 연구 결과물입니다. TimesFM 2.5라는 업그레이드 버전이 최근 공개됐고, 구글 BigQuery(구글의 대규모 데이터 분석 서비스)에 공식으로 통합되면서 실제 업무에 바로 쓸 수 있다는 점이 재주목을 끌었습니다.

### 핵심 기능

- **학습 없이 즉시 사용**: 별도의 AI 학습 과정 없이 바로 예측에 사용합니다. 데이터 과학자(데이터를 분석하는 전문가)가 없어도 예측 모델을 바로 돌릴 수 있습니다
- **긴 시계열 처리**: 최대 16,000개의 과거 데이터 포인트를 참고해서 최대 1,000개 미래 시점까지 예측합니다. 2년치 일별 매출 데이터를 넣으면 그 이후 예측을 뽑을 수 있습니다
- **공변량 활용**: 공변량(예측에 영향을 주는 부가 정보, 예: "이날은 연휴였다", "이달은 프로모션을 했다")을 함께 입력하면 예측 정확도가 크게 올라갑니다

<div class="example-case">
<strong>실전 예시: 중소 이커머스의 재고 관리 담당자</strong>

온라인 쇼핑몰 운영팀의 최지현 씨는 매달 재고를 얼마나 주문해야 할지 감으로 결정하다 보니 과재고로 창고 비용이 늘거나, 품절로 매출을 놓치는 일이 반복됐습니다. timesfm에 지난 2년간의 일별 판매량 데이터를 넣고 "다음 달 예상 판매량"을 요청했더니, 학습 없이 30초 만에 품목별 예측치를 반환했습니다. 여기에 "다음 달 대규모 할인 행사가 있다"는 공변량을 추가하니 예측 정확도가 더 올라갔습니다.

</div>

<div class="example-case">
<strong>이런 분께 추천해요</strong>

- **소매업·유통업 재고 관리자**: AI 학습 없이 과거 판매 데이터만 있으면 바로 재고 예측에 쓸 수 있습니다
- **스타트업 데이터 분석 담당자**: 전담 AI 엔지니어 없이도 구글 수준의 예측 모델을 업무에 바로 적용할 수 있습니다
- **금융·에너지 분야 애널리스트**: 주가, 전력 수요, 환율 등 시간 흐름 데이터라면 어떤 분야든 동일하게 적용 가능합니다

</div>

Claude Code를 혼자 쓰는 도구가 아닌 팀 단위 자율 개발 시스템으로 확장한다는 개념이 개발자들 사이에서 화제가 됐습니다. `ralph`, `ulw` 같은 매직 키워드(특정 에이전트 조합을 실행하는 단축 명령어)를 입력하면 복잡한 오케스트레이션이 한 단어로 실행되는 방식이 실용적이라는 평가를 받았습니다.

### 핵심 기능

- **19개 전문 에이전트 자동 협업**: 아키텍처(전체 구조 설계), 리서치(자료 조사), 테스트(기능 검증) 등 역할별로 나뉜 에이전트들이 자동 파이프라인(연속된 작업 흐름)으로 협력합니다
- **비용 최적화 모델 선택**: 작업 복잡도에 따라 Haiku(저비용·빠른 AI 모델)와 Opus(고성능·정밀 AI 모델)를 자동으로 선택합니다
- **스킬 자동 추출 및 재사용**: 반복되는 작업 패턴을 자동으로 감지해 스킬(재사용 가능한 작업 단위)로 저장하고, 다음에 비슷한 상황이 오면 바로 꺼내 씁니다

<div class="example-case">
<strong>실전 예시: 혼자 앱을 만드는 1인 개발자</strong>

프리랜서 개발자 도현 씨는 Claude Code에 기능 하나를 요청할 때마다 설계·코딩·테스트를 AI에게 순서대로 따로 시켜야 해서 시간이 오래 걸렸습니다. oh-my-claudecode를 붙인 후 `ralph` 한 단어를 입력하자, 설계 에이전트가 구조를 잡고 코딩 에이전트가 구현하고 테스트 에이전트가 검증하는 과정이 자동으로 돌아갔습니다. 같은 작업에 드는 시간이 절반으로 줄었고 AI 비용도 40% 줄었습니다.

## 10위. sherlock

| 항목 | 내용 |
|------|------|
| 만든 곳 | sherlock-project |
| 주 언어 | Python |
| 이번 주 ⭐ | +4,634 |
| 전체 ⭐ | 80,616 |
| 링크 | [sherlock-project/sherlock](https://github.com/sherlock-project/sherlock) |

### 이게 뭔가요?

특정 닉네임(사용자명)이 400개 이상의 소셜 미디어와 웹사이트에 존재하는지 한 번에 검색해주는 사이버보안 조사 도구입니다. 마치 **탐정이 사람의 디지털 발자국을 추적하는 것**처럼, 인스타그램·트위터·틱톡 등을 하나씩 직접 열어볼 필요 없이 명령어 하나로 어떤 플랫폼에 같은 이름의 계정이 있는지 한꺼번에 확인할 수 있습니다.

### 왜 이번 주에 주목받았나요?

전체 스타 80,600개 이상으로 이번 주 목록 중 역대 누적 스타 1위입니다. 디지털 포렌식(디지털 기기와 온라인 흔적을 수사하는 기술), 펜테스팅(보안 취약점을 미리 찾아내는 모의 해킹), 저널리즘 취재 등 다양한 분야에서 꾸준히 쓰이는 스테디셀러인데, 새로운 플랫폼 추가와 클라우드 실행 지원 업데이트로 다시 화제가 됐습니다.

### 핵심 기능

- **400개 이상 플랫폼 동시 스캔**: Instagram(인스타그램), Twitter(트위터), TikTok(틱톡) 등 400개 넘는 사이트를 동시에 확인해 특정 사용자명의 계정 존재 여부를 알려줍니다
- **다양한 출력 형식**: 검색 결과를 텍스트·CSV(엑셀에서 열 수 있는 표 형식 파일)로 저장하고, Tor(익명 인터넷 브라우저)나 프록시(접속 경로를 우회하는 서버)를 통해 익명으로 검색할 수 있습니다
- **클라우드 실행 지원**: Docker(프로그램을 어디서든 같은 환경으로 실행하게 해주는 기술)와 Apify(클라우드에서 자동화 스크립트를 돌려주는 서비스)로 내 컴퓨터 없이도 실행할 수 있습니다

<div class="example-case">
<strong>실전 예시: 프리랜서 계약 전 상대방 확인이 필요한 사업자</strong>

온라인으로 외주 계약을 맺으려는 소은 씨는 계약 상대의 닉네임이 수상하다는 느낌을 받았습니다. Sherlock로 해당 닉네임을 검색했더니 여러 사기 관련 커뮤니티에서 동일한 이름의 계정이 활동 중임을 확인했고, 계약을 진행하지 않아 피해를 막을 수 있었습니다.

</div>

<div class="example-case">
<strong>이런 분께 추천해요</strong>

- **보안 담당자/펜테스터**: 조직 내 임직원의 온라인 정보 노출 범위를 파악하거나, 모의 해킹 시 공격 대상의 계정 현황을 사전 조사할 때 표준 도구로 활용할 수 있습니다
- **탐사 저널리스트/조사 기자**: 취재 대상 인물이 어떤 플랫폼에서 어떤 이름으로 활동하는지 빠르게 파악할 수 있습니다
- **본인 계정 관리가 필요한 일반인**: 내 닉네임이 다른 사람에게 도용되어 쓰이고 있지는 않은지 직접 확인할 수 있습니다
