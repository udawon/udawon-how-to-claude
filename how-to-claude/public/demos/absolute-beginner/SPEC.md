# 우리동네 카페 - 사이트 기획서

이 문서는 `index.html` 파일의 완전한 설계 명세서입니다. 이 문서만 읽고 사이트를 동일하게 재현하거나 수정할 수 있어야 합니다.

---

## 1. 프로젝트 개요

- **사이트 이름**: 우리동네 카페
- **영문 부제**: Our Neighborhood Café
- **HTML title**: `우리동네 카페 - 예약`
- **목적**: 카페 소개 + 좌석 현황 확인 + 예약 접수 (단일 페이지)
- **타겟 레벨**: 왕초보 — 단일 HTML 파일, 빌드 도구 없음
- **언어**: `lang="ko"` (한국어)
- **기술 스택**:
  - HTML5 단일 파일
  - Tailwind CSS v3 (CDN: `https://cdn.tailwindcss.com`)
  - Vanilla JavaScript (인라인 `<script>`)
  - 외부 이미지/아이콘 없음, 이모지만 사용

## 5. 기능 명세

### 5-1. 폼 제출 동작

```javascript
document.getElementById('bookingForm').addEventListener('submit', function(e) {
  e.preventDefault();
  alert('예약이 접수되었습니다!');
  this.reset();
});
```

1. `submit` 이벤트 발생 시 기본 동작(페이지 이동)을 차단합니다.
2. `alert`으로 완료 메시지를 표시합니다.
3. 폼을 초기 상태로 리셋합니다.

**참고**: 서버 연동 없음. 데이터는 어디에도 저장되지 않습니다.

## 6. 반응형 규칙

이 사이트의 반응형 처리는 최소한으로 되어 있습니다.

| 요소               | 모바일 (기본)        | sm (640px 이상)        |
|-------------------|---------------------|----------------------|
| 폼 필드 그리드      | `grid-cols-1` (1열) | `grid-cols-2` (2열)   |
| 테이블             | `overflow-x-auto`로 가로 스크롤 허용 | 동일 |

- `max-w-3xl`은 768px이므로 대부분의 데스크탑에서 가운데 고정됩니다.
- 히어로의 영업시간/위치: `flex gap-6`이므로 매우 좁은 화면에서는 줄바꿈될 수 있습니다 (`flex-wrap` 미지정).

## 7. 수정 가이드

### 카페 이름 변경
다음 3곳을 모두 수정해야 합니다:
- `<title>` 태그 (line 6)
- `<h1>` 태그 (line 52): `☕ 우리동네 카페`
- 푸터 `<p>` (line 197): `☕ 우리동네 카페`
- 푸터 저작권 (line 200): `© 2026 우리동네 카페`

### 색상 테마 변경
`tailwind.config`의 `colors.coffee` 객체 값을 수정합니다 (line 13-24). 모든 `coffee-*` 클래스에 자동 반영됩니다. 예를 들어 파란색 테마로 바꾸려면 coffee 값들을 파란 계열 hex로 교체하면 됩니다.

### 좌석 추가
`<tbody>` 안에 `<tr>` 행을 추가합니다. 기존 행 패턴을 복사한 뒤:
- 테이블명, 좌석 수, 위치, 상태 텍스트를 변경
- 배경색을 교차 패턴에 맞게 설정 (`bg-cream-100` / `bg-cream-50`)
- 마지막 행이면 `rounded-bl-lg`, `rounded-br-lg`를 새 행으로 이동하고, 이전 마지막 행에서는 제거
- 이전 마지막 행에 `border-b border-coffee-100` 추가

### 영업시간/주소/전화번호 변경
- 영업시간: line 67의 `🕐 매일 09:00 - 22:00`
- 위치 (간략): line 68의 `📍 서울시 마포구`
- 주소 (상세): line 198의 `📍 서울시 마포구 연남로 42길 12, 1층`
